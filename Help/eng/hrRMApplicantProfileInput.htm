Field descriptions

“General” Information

Position Applied

Retrieved from Position List; with positions in the applied recruitment period only;
This position is also used in Recruitment Decision data entry and later reports.

Department Applied

Retrieved from Department List; with departments in the recruitment period and position that the applicant applied;
This position is also used in Recruitment Decision data entry and later reports.

Desired Salary

Enabled to manually input the salary that the applicant desired, without decimal number

Start Working Date

Manually input; being retrieved to “Recruitment Decision/ Join Date”

Recruitment Channel

Retrieved from Recruitment Channel list

Recruitment Source

Retrieved from Recruitment Source list

Referred by

Manually input

The “Recruitment Source” and “Referred by” information will be auto-updated to “Employee Information/ Working History/ Other” after the “Recruitment Decision” is input.

Address/ Mobile Phone/ Email

Manually input

“Personal” information

The personal information input in this function will be auto-updated to “Employee Information” after the “Recruitment Decision” is input.

“Health” information

The health information of applicant will also be auto-updated to “Employee Information” (Personal Information\Health) after the “Recruitment Decision” is input.

To the “Blood Type” field, you should use space bar to input.

“Education” information

Education information is used for reference only

“Working History” information

The working history of the applicant will also be auto-updated to “Employee Information” (Working History\Previous Working History) after the “Recruitment Decision” is input.

“Foreign Language” information

The foreign language information will also be auto-updated to “Employee Information” after the “Recruitment Decision” is input.

“Certificate” information

This information is used for reference only.

Attachments

The program allows users to add attachment files to a voucher. To add an attachment file, ckick on the icon then choose file. To download an attached file, ckick on the icon . To remove an attached file on a voucher, ckick on the icon .

- When entering "Recruitment Decisions", if Applicant profile have attachments, the program will copy all files to Employee Information;
- Program allows attachments up to 10 MB, the maximun size of each file can be declared. (Refer this function to counsellors)

Import data from excel spreadsheet

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:
- Enter “Applicant Profile Input”;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to grid view of the voucher.
Select type of copy as 1 - Overwrite if you want to update the available timekeeping data. If you select 0 - No, the system will insert required timekeeping data to blank only.

Notes of inputting data in template file:
- Period Code: must be one existing in “Recruitment Period Information Input”;
- Gender: input 1 or 2 (means male or female);
- Application Form Status/ Recruitment Channel/ Recruitment Resource: must be one existing in corresponding lists;
- Position Applied: must be positions in the applied recruitment period;
- Department Applied: must be departments in the recruitment period and position that the applicant applied;
- Desired salary, Mobile phone: input in number format without decimal number;
- Date of Birth, Date Applied and Start Working Date: input in date format.

The system will give error message in some following cases:
- Data in column containing object codes do not exist in the corresponding lists. In this case, the system will indicate the specific invalid cell;
- String data is longer than the available length of field (indefinite error);
- There are at least 2 records with the same applicant ID;
- Does not exist one in following information: period code, applicant ID, full name, date of birth, place of birth, gender, date applied, application form status, position applied, department applied, desired salary, start working date, recruitment channel, recruitment source, mobile phone.

Other notes:
- You must have the "New" right in “Applicant Profile Input” to import new data from spreadsheet to detail grid;
- You must have the "Edit" right in “Applicant Profile Input” to edit the imported data (over-written).
