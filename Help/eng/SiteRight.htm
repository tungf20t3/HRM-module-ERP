Access Right by Site

Access Right by Site used to restrict access or edit the data of an unauthorized warehouse. If not authorized the "Access" right to a warehouse, the current user can not see data the report as well as editing of the data.

- In voucher screen has "Current Stock" field, the program will not show this data;
- In "View" or "Search" screen (voucher), or the browser windows of report query, the program will not show data of records that contain the sites not authorized the "Access" right.

Notes:
- Only Administrator can execute this function;
- This function will not apply on user group;
- This function will not apply on the cost calculating processes;
- Also, this function will not apply on import data from excel spreadsheet utilities;
- If no declaration has been added, it means user has full right in all sites of the authorized units;
- This function will not apply on reports query screens, but result browser can only show the data of the sites authorized the "Access" right;
- In the financial and accounting reports or books with item in detail option, the program will not apply the "Right by Site", but when drill-down.
