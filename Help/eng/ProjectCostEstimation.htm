Input Forecast Amount

Besides inputting opening balance and accumulative amount from starting work, user must input forecast amount for job/project in a specific screen only once when beginning to use the program. It will be automatically transferred to next years.
Path: Module → Investment Accounting --> Data Entry → Input Forecast Amount

Entry screen

Field descriptions

Year

Forecast year of project, work details

Project

Code of project/job having forecast

Amount/FC Amount

Forecast amount of project/job
Reports:

    - Investment Implementation by Project, Work, Work Detail;
    - Investment Liquidation by Project, Work, Work Detail.
