Employee Requirement Input

This function is used to input the number of employees that each department requires; to be used for “Vacancy Status Report"

Department

Only departments belonging to the assigned unit

Position

This is a required field, retrieved from Position List.

No. of Employees

Used to compare with the current number of employees and calculate the different number

Effective Date

When getting data for "Vacancy Status Report", the system will select the record with the latest “Effective Date” before the “Report Date”.

Import data from excel spreadsheet

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:
- Enter “Employee Requirement Input”;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to grid view of the voucher.
Select type of copy as 1 - Overwrite if you want to update the available timekeeping data. If you select 0 - No, the system will insert required timekeeping data to blank only.

Notes of inputting data in template file:
- Department Code: must be the assigned departments with the same length in add (edit) screen in list;
- Position Code: must be one existing in the Position List;
- No. of Employees: input in number format, 6 characters maximum without decimal numbers;
- Department, Position Description, Note: user-defined information;
- Effective Date: input in date format.

The system will give error message in some following cases:
- Data in column containing object codes do not exist in the corresponding lists. In this case, the system will indicate the specific invalid cell;
- String data is longer than the available length of field (indefinite error);
- There are at least 2 records with the same department, position and effective date;
-Does not exist one in following information: department, position or effective date.

Other notes:
- You must have the "New" right in “Employee Requirement Input” to import new data from spreadsheet to detail grid;
- You must have the "Edit" right in “Employee Requirement Input t” to edit the imported data (over-written).
