Input Opening Quantity of WIP

Entry Screen

Field Descriptions

Factor Type

Declaration of WIP type by factor

Product

Code of product WIP opening

Department

Code of cost center in case product WIP by cost center. This is a required field if cost is calculated by department in system parameter.

MO Number

Number of MO WIP in case product WIP in MO. This is a required field if cost is calculated by MO in system parameter.

WIP Quantity / Finish Rate / Exchange Quantity

They are quantity of product WIP and quantity of product after exchanged to finished product by exchange rate.
If WIP rate is 50%, finish rate will be entered 50.

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.

The data importing process:

- Enter "Input Opening Quantity of WIP" menu;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to balance file in the system.

Notes of inputting data in template file:
- Factor Type, Product, Department, MO Number: must be one existing in the corresponding lists (Cost Factor List, Item List, Section List, Manufacturing Order) with the same length in add (edit) screen in list;
- Department, MO Number: are optional fields depending on structure of the costing problem;
- WIP Quantity, Finish Rate, Exchange Quantity: number format, 12 digits and 4 decimal digits maximum (standard format);
- Product Name: user-defined information, 128 characters maximum.

Other notes:
- Cannot import data after system data closing date (the data closing date after the beginning input date);
- The current user must have the "Edit" right in this function to update the available data in the system (Copy type 1 - Overwrite);
- The current user must have the "New" right in this function to add new data to the system (Copy type 1 - No);
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
