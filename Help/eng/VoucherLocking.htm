Data Closing by Voucher

Path: System → Data Maintenance → Data Closing by Voucher
This function helps to close data of each voucher and by user.
Data up to closing date cannot be added or edited. To edit data in a closed period when necessary, it is required to change the closing date to the previous period.

For example:
To open 05/2012 period, set closing date as 30/04/2012 (period 04/2012).

In case of single locking (each user and voucher), only add new line then input user, voucher, and locking date.

Addvanced Locking

In case of multi-locking (all user and many voucher), user should use "Addvanced Locking" to save hours of manual effort.
Click on the icon to open the following screen:

Then, input the list of voucher code and locking date, the program will lock these vouchers with all user.
