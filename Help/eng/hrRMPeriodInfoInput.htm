Recruitment Period Information Input

Entry screen

Some special fields

Position Code/ Department Code

Position code must be one defined in the examination form input.
This position/ department code will be used for applicant profile input, in accordance with recruitment requirement by department and position.
Only departments in current unit are available to be selected.

Quantity

Being displayed in Recruitment Requirement report

Total

This is total number of required employees in the grid; will be displayed in Recruitment Requirement report.

“Retrieve data from Recruitment Requisition Form”

- Request Date from: this is a required field;
- Request Form No.: this is a required field; only request forms in the current unit with the “Request Date” after or the same as the “Request Date from” in the filter condition will be displayed. When you click “OK” the following screen appears:

If you click “OK”, the system will get positions being checked in “Select” option only to grid. Each position of each department in “Recruitment Requisition Form” can be retrieved once.
