Location List

Path: Modules → Inventory Management → List Maintenance → Location List


Location list is designed for stock controlling by location purpose; used in data entries relating to stock.
To sites having locations, it is required to select location code in data entries such as:

    - Item Opening Balance
    - Purchase Invoice
    - Purchase Expense Transaction
    - Purchase Receipt – Direct Issue Voucher
    - Return to Supplier
    - Sales Invoice
    - Sales Return
    - Receipt
    - Material Requisition
    - Issue
    - Stock Transfer out
    - Stock Transfer in

Note:
A site may consist of many locations, so site code can be used again. However, location is unique.

See “Hints and Tips in List Setting” for details of common tasks in list maintenance
