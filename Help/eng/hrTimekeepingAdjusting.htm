Timekeeping Data Adjusting

Entry screen

Field descriptions

Add Item

Timekeeping data shown in this function are from following cases:

    - Data are system-generated from employee workday computation. You can adjust the time in/out or “next day” option in this function. Refer to Data for Workday Computation in for more details.
    - If there is no timekeeping data, click to add new item.
    - Besides, if you have any information adjusted in both the generated and original data, there will be one more original row in this function.

Note:
All available data rows in “Timekeeping Data Adjusting" will be shown in timekeeping report after workday computation.

For example:
Timekeeping data of employee NV001:
- In "Employee Time In/Out Adjusting":
+ Time in: 08:00
+ Time out: 17:00
- After workday computation, in “Timekeeping data adjusting”:
+ Time in: 08:00
+ Time out: 17:00
Edit to:
+ Time in: 08:05
+ Time out: 17:05
- Add new row in “Timekeeping data adjusting”:
+ Time in: 08:10
+ Time out: 17:10
- In “Employee Time In/Out Adjusting”, edit the:
+ Time in: 08:15
+ Time out: 17:15
Process the “Workday Computation” again, the result will be:
- In “Timekeeping data adjusting”:
+ Time in: 08:05 and Time out: 17:05
+ Time in: 08:10 and Time out: 17:10
+ Time in: 08:15 and Time out: 17:15
- In timekeeping report of employee NV001, there will be the same data rows.

Delete

When checking in “Delete” check-box and saving, there are following cases occurring:

    - To data row input after workday computation: the system will check in delete check-box. It will disappear in report after having workday computation run again.

Note:
+ You can uncheck the delete checkbox, then the data row will appear in report after having workday computed again.
+ This data row will really be deleted when it is deleted in original data.

    - To data row input in “Timekeeping data adjusting”: when checking “delete” and saving, it will be deleted and not appear in report after you have workday computed again.

Save

When you have timekeeping data adjusted, click to save the changes.
