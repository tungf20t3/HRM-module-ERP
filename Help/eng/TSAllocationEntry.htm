Tool & Supply Allocation Entry

You can create Tool & Supply Allocation Entry to post to general ledger.
Path: Modules → Tools & Supplies Management → Data Entry → Tool & Supply Allocation Entry

Entry screen

Allocation entries are created when creating Tool & Supply List. Program will group entries by allocation account and reference expense account from detailed allocation information of tools & supplies.

Operations

Create entry

Click - “Create” to create a new entry

Delete entry

Click - “Delete” to delete created allocation entries. They can be only deleted in this function, not in Journals.

Print

Click - “Print” toolbar icon to print created allocation entries
