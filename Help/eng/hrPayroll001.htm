Payslip

Field descriptions

Number

Ordinal number of the article in definition and in report

Code

Code of the article supporting in data process

Description

Name of the article shown in report

Other Name

Name of the article in English

Font Bond

    - If 1- Bold, this article name will be in bold font;
    - If 0- Regular, this article name will be in regular font.

Print

    - If 1- Print, this article will be shown in report;
    - If 0- Not Print, this article will be hidden.

Calculating Way

    - By Generated Data: data will be get from salary summary, salary detailed by product or allowance file;
    - By Formula: the calculation of the article can be defined in the “Formula” field below.

Source

Data of articles calculated by generated data will be get from: salary summary, salary detailed by product, allowance detailed sheet, salary detailed by workday, award and penalty detailed sheet, other income/deduction detailed sheet and workday detailed sheet.

Data Type

There are 2 options: 1 - Amount, 2 - Product Quantity. The option 2 is used only when data source is from "Salary Detailed by Product".

Departments

Working departments of employee; extracted from Department List

Pay Types

Extracted from Pay Type definition function

Products

This field is enabled to select (from Product List) only when the data source is from "Salary Detailed by Product" file.

Allowances

This field is enabled to select (from Allowance List) only when the data source is from "Allowance Detail" file; used for pay type P20 - Allowance.

Formula

This field is used to define the formula to calculate value of article in case the calculating way is 2 - By Formula.
