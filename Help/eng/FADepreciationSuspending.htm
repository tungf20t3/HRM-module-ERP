Fixed Asset Depreciation Suspending

The program allow to suspend the depreciation in a duration for Fixed Assets depreciated by "Straight-line" method.

Notes:
- If value of "FA Depreciation Calculated by Day" parameter is "No", the program will not depreciate for the month existed the start suspending declaration;
- If value of "Depreciate the Whole Remaining Amount in Last Period" is "Yes": The depreciation will not be applied for the month existed the end suspending declaration;
- The suspending durations must not be overlapped or duplicate for an asset code.
