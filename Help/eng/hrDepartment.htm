Department List

Field descriptions

Parent Department

used for organization structure appearance

Default Employee ID

It is defined to be added to employee ID as a prefix when creating a new employee ID in this department. This field is defined to each department. If a department is not defined "Default Employee ID", it will be get one in parent department (if any).

Cost Center

Extracted from Department (Section) List, used to auto-post department cost in salary accounting.

Unit

Extracted from Unit List; this is the unit in which the current user has the right to access all departments. The unit of a department is default to be the same as one of parent department.
