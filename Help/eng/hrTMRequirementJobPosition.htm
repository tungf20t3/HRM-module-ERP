Training Requirement Input by Job Position

Entry screen

The information input in this function will be used for Training Requirement Report by Job Position.

Import data from spreadsheet (excel)

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:

    - Enter “Training Requirement Input by Job Position”;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to grid view of the voucher.

Select type of copy as 1 - Overwrite if you want to update the available timekeeping data. If you select 0 - No, the system will insert required timekeeping data to blank only.

Notes of inputting data in template file:
- Department: must be assigned one with the same length in add (edit) screen in list;
- Position, Subject: must be one existing in Position List, Subject List;
- Request Date from/to: input in date format.

The system will give error message in some following cases:
- Data in column containing object codes do not exist in the corresponding lists. In this case, the system will indicate the specific invalid cell;
- String data is longer than the available length of field (indefinite error);
- There are at least 2 records with the same both department, position and subject;
-Does not exist one in following information: department, position, subject or effective date.

Other notes:
- You must have the "New" right in “Training Requirement Input by Job Position” to import new data to voucher detail grid;
- You must have the "Edit" right in “Training Requirement Input by Job Position” to edit data in list.
