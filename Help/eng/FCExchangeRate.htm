Exchange Rate List

Path: Modules → System → Exchange Rate List
This function is used to input and save timely exchange rate of all foreign currencies.



In entries of arising in foreign currency, the default exchange rate will be the latest one (towards the transaction date) extracted from this list.

For example: the exchange rate of USD on 01/01/2010 is VND 19,500; on 01/10/2010 is VND 21,500.
Creating Receipt in USD on 01/02/2010, the default value of exchange rate field will be 19,500.
Creating Receipt in USD on 15/10/2010, the default value of exchange rate field will be 21,500.


See “Hints and Tips in List Setting” for details of common tasks in list maintenance
