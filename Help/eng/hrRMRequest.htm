Recruitment Request Form

This is the first step in the recruitment process, used to input the requirement in “Recruitment Period Information Input” and used for later reports.

Entry screen

Field descriptions

Format view

Department

This is a required field with departments in the current unit only.

Requested by

The person who requires the requisition; optional field

Content

Note of the requisition; optional field

Total

Total “Quantity” in grid; reference information and un-editable

Grid view

Position Code

Retrieved from position list; required field

Quantity

This is a required field without decimal number and must be greater than 0.

Required skills

The required skills for the required position

Request Date from/to

Required fields; is used as filter condition for “Retrieve data from Recruitment Requisition Form” in “Recruitment Period Information Input".
