Request for Quotation

“Request for Quotation” screen

Field descriptions - Format view – upper area

Supplier ID

Extracted from “Supplier/Customer List”, required field.

Requisitioned by

The person requires or makes the Request for Quotation.

Description

The reason, purpose of request for quotation

Due Date

Date when supplier must give quotation

RFQ No.

Number of Request for Quotation can be entered by user or automatically numbered by program following declaration in Transaction List.

RFQ Date

Date when Request for Quotation is created

Status

Status of RFQ is automatically processed by program. There are 2 statuses: “No Action” and “Open”.

Field descriptions – Grid view

Item Code

Code of items in RFQ, extracted from “Item Master” (it required field)

UOM

Default UOM in “Item Master”, can be changed by conversion of units of measurement in some special cases.

Quantity

Quantity of items required quotation (it is required field).

Note

Other notes related to items required quotation.

Job

Job relating to RFQ
