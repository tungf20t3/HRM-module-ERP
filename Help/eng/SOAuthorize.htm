In operating Open SO, Process Dept. can:

    - Open all SO
    - Cancel SO - Cancel the previous approval

After creating Sales Order, you can select action of SO as “Open” which can be carried out at SO creating step as presentation. This section gives guides of canceling “Open” SO.
In this section, program allows user to “Cancel Approval” of approved SO in order to support adjustment in previous steps.
SO being suspended can be process by function “Operation Assistant”.
SO can be saved in “Open” status by 02 ways:

Hold by users

Select status “Hold” and save voucher. Direct suspension can be applied to transaction code 1 – SO and transaction code 2 - Contract

Auto-hold

When selecting status “Open” and save voucher, program will check receivables from customer at that time. Auto-hold can only be applied to transaction code 1 – SO. Receivable amount is checked basing on credit limit of customer declared in Customer/Supplier List and on value of contracts which are signed, in process or finished but haven’t paid by customer.

Operation Assistant screen

Tasks to cancel “Open” status of SO

Open "Operation Assistant" function, system will display the list of "Holded SO". Check on “Select” option, then click “ - Approve” to save the result. This SO line will be moved out from browser screen and changed status to "Open".

Tasks to cancel Approval of SO

If it is required to cancel result of SO which have been approved, you must run Cancel Approval function.
In “Operation Assistant” screen, click “ - Search”, then choose the status "1 - Approved" in tab "Other" to filter approved SO.
In “Approved Vouchers” screen, program lists all “Approved” vouchers which satisfy filter condition.
Check on option “Select”to select SO needs to be cancelled approval.
Click - “Undo” to create new status. Program will change this SO status into “Open”.

Tasks to cancel an "hold" SO

Open "Operation Assistant" function, system will display the list of "Holded SO". Check on “Select” option, then click “ - Cancel” to save the result. This SO line will be moved out from browser screen and changed status to "Cancel".
