Return to supplier

Field descriptions - format view

Supplier

Supplier ID and name; extracted from “Supplier List” (required field).

Receiver

Person receives goods, free entered.

Transaction Code

Default 1 - Return to Supplier

Reason Code/ Debit Account

Reason code; extracted from “Reason List”. Debit account is payable account of supplier, extracted from “Supplier List”.

Invoice No.

Number of “Return to Supplier” invoice; user-defined or system-generated

- If the invoice is numbered by voucher book which has been defined the "Invoice Form", this "Invoice Form" will be used in tax reports.
- If the invoice is numbered by voucher book which has not been defined the "Invoice Form", the system will get the "Default Output Invoice Form" in optional parameter.
- If the "Default Output Invoice Form" parameter is also blank, there will be no "Invoice Form" in tax reports.

Serial No.

Serial number of return to supplier invoice

Inv. Date / GL Date

The date the Return to Supplier invoice is created and the date all entries in invoice are posted to GL.

Ex. Rate

Select FC code and declare exchange rate in date of creating transaction.

Status

There are 3 statuses:

    - No Action: the voucher has not been posted to any book;
    - Release Later: If there is VAT data, the invoice will be shown in VAT reports. If this is AR/AP invoice, it will be shown in AR/AP by invoice reports;
    - Release: the voucher has been posted to all related books.

Field descriptions - “Details” tab

Item code

Item code in voucher. It can be directly entered or extracted from Receipt, QA Transaction or Invoice.

UOM

Default unit of measurement in “Item Master”. It can also be changed by conversion of units of measurement in some special cases.

Site

Site issuing item

Location

Location of items issued, used in case you wish to control items by location.

Lot

Lot code of returned items used in case you wish to control items by lot.

Quantity

Returned quantity (required field).

Price

Issuing unit price of item in GL currency

Amount

Amount of item in GL currency

Credit Account

Item account.

Field descriptions – “Input VAT” tab

Report Form

Report forms are used as filter condition of reports in Tax Reports module;
Consist of:

    - 3 – VAT Invoice (default);
    - 4 – Goods, services purchased without invoice;
    - 5 – Common sales invoice.

Tax Type

VAT types of purchased goods, services; used to classify taxes when reporting “Input VAT Invoice Listing” (Form 01-2/VAT)”.
Consist of:

    - 1 - Goods, services used for VAT taxable business activities with revenue;
    - 2 - Goods, Services used for VAT non-taxable business activities;
    - 3 - Goods, Services used for both VAT taxable and non-taxable business activities;
    - 4 - Goods, services used for investment projects which meet tax deduction condition;
    - 5 - Goods, services not be declared in Form No. 01/GTGT.

Tax Code

Code of VAT; extracted from VAT list

Tax Acct.

Account of VAT; in accordance with tax code but editable

FC Tax

Total VAT amount in foreign currency (appears only when transaction is in foreign currency)

Tax

Total VAT amount in accounting currency

Tax Authority

State authority to which your company has tax declared

Item group

Name of goods returned; for input VAT invoice listing purpose

Note

Number, date of purchase invoice; for input VAT invoice listing purpose

"Post to Input VAT" option

When this option is checked, the system will check tax information and post to Input VAT listing. Otherwise, tax information of this voucher will be posted to Output VAT listing (as default).
