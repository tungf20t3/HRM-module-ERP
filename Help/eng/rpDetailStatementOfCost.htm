Detail Statement Of Cost Report

This report is built by:

    - Date, item: The program will store the report data by report day and item in filter form.
    - Manufacturing Order, Department: Consolidate with Report Date and Item, these fields (Manufacturing Order, Department) will also generate a new report. However, its depend on the declaration of CO parameters.

Note:
Choose "Get data from BOM" as "1 - Yes" when you want to update all the changes of BOM to existing report.
