Training Course Information

Entry screen

Some field descriptions

Format view – upper area

Course Code

This is required field with 16 characters maximum.
The code is systematically increased. For example: the current code is "Khoa0014", key in "Khoa00"and press down arrow key, the new code "Khoa0015" will appear.

Status

There are 5 statuses: 0 – No Action, 1 - Planned, 2 - Opening, 3 – Completed and 4 - Cancelled.
Default status is defined in “Voucher Information” but editable to status 0, 1, 2 and 4. The status 3 – Completed will be system-generated after inputting “Training course result”.

Note:
- If the default status in “Voucher Information” is 3 – Completed, when you add new “Training Course Information”, the system will change to status 0 – No Action.
- Cannot edit the “Completed” training courses; if you edit, the system will give message “You are not assigned to this function or access deny. Contact the administrator if you have the right to access this function.”

Grid view

There are 3 ways to input information in grid:

    - Add new (Ctrl+Insert);
    - Retrieve data from Training Request Form;
    - Import data from spreadsheet (excel).

Retrieve data from Training Request Form

- Request Date from: required field;
- Request Form No.: required field; only Request Forms with the “Request Date” (date created) ≥ before or the same as the “Request Date from”, belonging to the same unit with the current record and having the same course code with the current record. After clicking “OK”, the following screen will appear:

If you click “OK”, the system will get information of employees being checked in “Select” option only.
Each employee in “Training Request Form” can be retrieved to “Training Course Information” once.

Import data from spreadsheet (excel)

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:

    - Enter “Training Course Information”;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to grid view of the voucher.

Notes of inputting data in template file:
- Employee ID: must be unique one with the same length in add (edit) screen in list; only employees belonging to departments in the assigned unit;
- Department, Position: the current department and position of the employee.

The system will give error message in some following cases:
- Data in column containing object codes do not exist in the corresponding lists. In this case, the system will also get data to grid but will give error message when saving. You have to edit the information before saving;
- String data is longer than the available length of field.

Other notes:
- The function is only supported to get data for each voucher, used in case of frequently inputting the similar periodic transactions.
- You must have the "New" right to import new data to voucher detail grid;
- You must have the "Edit" right to edit data in voucher detail grid;
- It is not required to keying object name in lists, the system will auto update following their codes.

Resources

This is reference information, extracted from “Training Resources” definition.

General

Training Date from/to

Training date from/to is default as Plan Date from/to but editable. This information will be used in “Training Course Result” entry and “Training Implementation” report…

Attachments

The program allows users to add attachment files to a voucher. To add an attachment file, ckick on the icon then choose file. To download an attached file, ckick on the icon . To remove an attached file on a voucher, ckick on the icon .
Program allows attachments up to 10 MB, the maximun size of each file can be declared. (Refer this function to counsellors)

Other

The information in this tab will be shown in Training Plan, Training Implementation reports…

Total Planned Cost

Total planned cost is default to be equal to Planned No. of Trainees multiplied by Planned Cost per Trainee; but editable.

Total Actual Cost

Total actual cost is default to be equal to Actual No. of Trainees multiplied by Actual Cost per Trainee; but editable.

Company Support (%)

Be default as 100 but editable; cannot input negative number or number over 100.
