Guides of online approval function

Approve via mobile phone

To use this function, it is required to register 3G service and 3G USB device. To relevant configuration, please refer to consultant during implementation process.
How to use:
Select the order in “Open” status, input approver code and save the order. The system will send the approval request to the approver's mobile phone with brief information including order's number, memo, total amount, approval code and cancel code. The approving user simply replies to the message with approval or cancel code, the system will change the order status to “Approved” or “Closed”.

For example:
The approving user receives a message with following information:

"Order#: NC4-01-12
Total Amount: 24 000 000
Approval Code: 7529D7
Cancel Code: 77AA9F""

To approve the order, reply with ""7529D7""; if not, reply with ""77AA9F""

Approve via e-mail

To use this function, assume that the company has registered the internet service; to relevant configuration, please refer to consultant during implementation process.
How to use:
Select the order in “Open” status, input approver code and save the order. The system will send the approval request to approving user’s e-mail. The user simply clicks on “Approve” or “Cancel” link to approve or cancel, the system will change the order status to “Approved” or “Closed”.

Some common operations in Import Purchase Order

Create Import Purchase Order by Extract data from PO

Create new Import Purchase Order, click " - Extract Data”, select “Extract data from PO” menu, Input the date from which the PO is selected, then click to lookup PO. Only PO in “Approved” or “Receipt” status of the selected supplier appears in the screen. Select the required PO, input the order quantity and exit the PO list screen. The selected rows in the PO list will be systematically updated to Import Purchase Order.

Create Import Purchase Order by Extract data from Purchase Requisition

Create new Import Purchase Order, click " - Extract Data”, select “Extract data from PR” menu, Input the date from which the PR is selected, then click to lookup PR. Only PR item in “Approved” status appears in the screen. Select the required PR, input the order quantity and exit the PR list screen. The selected rows in the PR list will be systematically updated to Import Purchase Order.

Create Import Purchase Order by Extract data from Supplier Allocation

Create new Import Purchase Order, click " - Extract Data”, select “Extract data from Supplier Allocation” menu, Input the date from which the Supplier Allocation is selected, then click to lookup Supplier Allocation. Select the required Supplier Allocation, input the order quantity and exit the Supplier Allocation list screen. The selected rows in the Supplier Allocation list will be systematically updated to Import Purchase Order.

Working with attachments

The program allows users to add attachment files to a voucher. To add an attachment file, click on the icon then choose file. To download an attached file, click on the icon . To remove an attached file on a voucher, click on the icon .
Program allows attachments up to 10 MB, the maximum size of each file can be declared. (Refer this function to counsellors)
