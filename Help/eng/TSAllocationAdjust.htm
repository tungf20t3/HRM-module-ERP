Adjust Monthly Allocation

If you want to change distribution of a month but do not want to change ones of next months, use “Adjust monthly distribution” function.
Path: Modules → Tools & Supplies Management → Adjust Monthly Allocation

Entry screen

Field Descriptions

Tool & Supply Code

Code of adjusted tool/supply, extracted from Asset List

Period/ Year

Month/year of distribution adjustment

Edit Allocation

Select edit to input adjusted value

Currency Code

Currency code is extracted from tool & supply information list.

Ex. Rate

Exchange rate is extracted from tool & supply information list by each tool & supply code.

Original cost / Allocated Value/ Current Value

Reference information, un-editable
Add to foreign currency fields for tools & supplies having foreign currency value.

Monthly Allocation

Input adjusted value; can only be input if “Edit allocation” field is “1”
