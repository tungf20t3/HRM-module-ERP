Examination Form

Entry screen

Some field descriptions

Form Code

Examination form is defined for each required position. Each examination form consists of many tests and interviews.
Examination form is used in data entry as well as filter condition for reports.

Required Position

Retrieved from “Position List”

Round Code

Input all test and interview rounds in an recruitment examination

Content

The content of the recruitment examination; which will be retrieved to format view of Test or Interview Result Input

Passed/ Failed Status

These fields are retrieved from “Application Form Status”, will be used to update the status in “Applicant Profile Input” after inputting recruitment examination (test and interviews) by recruitment round.
