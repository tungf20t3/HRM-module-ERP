Hints and Tips during AI ERP Processing

Usable Browsers

The browsers and versions recommended for AI ERP running well; however, you can upgrade to newer versions.

    - Internet Explorer 8.0 (or higher)
    - Firefox 3.6 (or higher)
    - Chrome 9.0 (or higher)
    - Safari 5.0 (or higher)
    - Opera 11.0 (or higher)

Browser Settings

FireFox – Incorrect display stretching

When using FireFox, if the display stretching is incorrect (as the above image) – because of FireFox error – adjust browser options as following steps:

    1. Enter Menu/Tools/Options/;
    2. Select Advanced/Browsing;
    3. Uncheck “Use hardware acceleration when available”;
    4. Exit FireFox and open it again.

Using the Unicode character set

When using Vietnamese keyboard or keyboard driver for the task, we recommend you always set the "Character Set" to Pre-compound Unicode.
In case of using Compound Unicode, some display errors will appear when print to pdf file.

Example
Charater “a” will be displayed as “ạ” at all.

Use “Check item code” function

In some browsers having “Check spelling” function as Chrome, it is required to disable this function for the “Check item code” function to process correctly. This is an optional function.

Common tasks in AI ERP

Login a working session

AI ERP is designed as web application; therefore, a browser, in a using period, is considered as a working session. In case of temporarily stop working, you should sign out to exit the working session. Otherwise, the system will auto disconnect in a short time (defined in system configuration). At that time, you have to sign in again to continue working.

Note:
A user can only sign in AI ERP once.
- If the user (already sign in) wants to sign in another computer (or browser), it is required to sign out;
- Closing the browser does not mean signing out the system. At that time, you have to wait for working session finish (now defined 20 minutes) to sign in again;
- In a client, only 1 user can sign in. If that user wants to sign in another client (or browser), the system will ask if that user want to sign out the current session or not. If yes, the previous session will be completely invalidated.
- Remember to check Caps Lock button when keying password because the login password is case-sensitive.
Besides, check Vietnamese typing program if it is configured as requirement (according to each typing program, user’s typing way).

In case “Number of days required to change password” defined in system optional parameter is over, the system will require user to change password.

Reset Password

In case of forget password, click on “Forgot Password?” in Login screen to reset password.

Enter your Account ID as “Name” and your Email Address defined in User Maintenance as “Email”.
If you can not see the characters in the picture, click on “Reload” to get a new one.

Click on “OK”, if you enter a valid Account ID and Email Address, you will receive a link to retrieve your password online via email after submitting this form.
The program will release a message: “Please check your email, We have sent you an email with a link for you to reset your password. Don't forget to also check your Junk/Bulk folder in case you don't get our email on time.”
A temporary password will be sent to user via email. Check email and follow the directions.

Note:
Password retrieved is temporary. Follow the directions to change password before logging in the system.

Login English interface

To work in English interface, when log in AI ERP, you can select “Tiếng Việt” or “English”.

During working session, if it is required to change, click “Sign Out” at the top right of the working screen.

Open New Window/Tab

To process many tasks at the same time (multi-task) when necessary, you can:

    - Copy the address of current page, open a new browser tab or window, paste it in the address field and open it;
    - In FireFox, Chrome and Opera: right-click on menu and select “Open in New Tab” or “Open in New Window”;

    - In IE (Internet Explorer): right-click on the menu icon and select “Open in New Tab” or “Open in New Window”

Moving cursor

AI ERP support the Enter and Tab key to facilitate the operation of the users who are familiar with Windows interfaces.

Working with code fields

In fields required codes from list, if you do not type the "like" value or type the incorrect value, then press "Enter", the system will show the whole items in list. If you type the "like" value, the system will show the items which meets the "like" string only.
Press Page Up or Page Down to search to required item, press "Enter" to return code value to inputting field.
In case you have not typed any character, press Page-down to show all items in list.
To optional fields, you can press "Enter" to pass.

Note:
To "Autocomplete" function (in list lookup), the priority is to search in "Description/Name" field, then in code field. It follows the "contain" rule because all code and name fields with the search "string" will be shown in the result. If you want to search for the code beginning with the keyed-in string, key in the required string and % mark.

For example: If you want to look up customer ID beginning with KH, key in the "customer ID" field the string "KH%".

Copy a paragraph or character

To copy a paragraph in web interface, right-click on the desired object, select cut, copy, paste or delete
This function should only be used in some browsers without Ctrl + C and Ctrl + V functions (as some browsers in cell phone)

When working in PC, it is more convenient to use hot key Ctrl + C, Ctrl + V.

View the complete titles of columns in grid

The titles of columns in grid may be so long, when we reduce the column width, some letters disappear and are replaced by three dots (…). To view the complete titles, scroll to the three dots “…” mark.

Calculation in number fields

In AI ERP, we can make the simple calculation; place the cursor at a number field and press the “+ - x /” operations to activate the calculation; when having the desired number, press enter.

Common tasks in date fields

To input the date in data entry, we can directly key in or select from the system by clicking on the icon at the right of date fields.

In the calendar screen , we can select the day only. To choose the month, use the left and right arrows. In case of selecting in many months and years, click on “month, year” area (at the middle of 2 arrows), we can select the year. After that, we can select the month. Thus, at the date field, we can select by following order:
01. Date, month, year
02. Month, year
03. Year - 10 years
04. Month, year
05. Date, month, year


Calculate in number field

The program allows simple calculating between the numbers. Put cursor in number field, press math “+ - x /” to activate calculation function. When has final result, press enter.

Resize object

The program allows scaling screen or fields in the grid to make things look better.

Expand and Restore Voucher Screen

You can simply expand voucher screen to entire screen and restore it by double-clicking on voucher title bar.
If the screen is maximized, double-clicking the title bar restores it. If it is not maximized, double-clicking the title bar maximizes it.

Moving between Vouchers

You can simply moving between voucher in view screen by click on the icon or in voucher title bar to move to the next or previous voucher.

Keyboard Shortcuts

The program also support alternate keyboard shortcuts. The buttons can be accessed by keyboard shortcuts by pressing and holding the Alt key + the corresponding underlined letter.

    1. Keyboard Shortcuts in Vietnamese interface: Mới [M], Sửa [S], Lưu [L], Nhận [N], Hủy [H], Đóng [g];
    2. Keyboard Shortcuts in English interface: New [N], Edit [E], Save [S], Ok [O], Cancel [C], Close [l];
    3. When working in the screen or form with multi tabs, the Tabs can be accessed by keyboard shortcuts by pressing and holding the Alt key + 1,2,3 (tab 1, tab 2, tab 3);
    4. Use Keyboard Shortcuts to move the focus to the fields under detail grid (voucher): The text box can be accessed by keyboard shortcuts by pressing and holding the Alt key + the corresponding underlined letter.

Guide to customize keyboard shortcuts for Firefox browser: use the [ALT] instead of [Alt] [Shift]
1. Open Firefox browser, type "about:config" in address bar;
2. Search "ui.key.generalAccessKey";
3. Change the value from -1 to 18;
4. Close and open Firefox again.
