Charge List

Path: Modules → Purchasing Management → List Maintenance → Charge List
Charges are defined to be used in purchase invoice, purchase expense transaction data entries.

Field descriptions

Charge Type

Extracted from Charge Type list; there are 3 types of charge now: insurance charge, freight charge and others.

Apportion by

Select to apportion charges to goods, services purchase/sales price by 1 – Quantity, 2 – Volume, 3 – Weight and 4 – Value

Example 1: Apportion by quantity
In Charge List: select charge code “BD – Loading charge”, Apportion by “1 – Quantity”
Create Purchase Invoice with 2 items: VT1 quantity as 100 and VT2 quantity as 200. In “Charges” tab, select charge code “BD” and input amount 15.000.000. Charge allocated to VT1 is [15.000.000 / (100 + 200)]*100=5.000.000; to VT2 is [15.000.000 / (100 + 200)]*200=10.000.000.

Example 2: Apportion by volume
In Charge List: select charge code “VC – Freight charge”, Apportion by “2 – Volume”
In Item List - “Size/Style/Color” tab: item VT1, input “Volume” as 1; item VT2, input “Volume” as 2.
Create Purchase Invoice with 2 items: VT1 quantity as 200 and VT2 quantity as 200. In “Charges” tab, select charge code “VC” and input amount 18.000.000. Freight charge allocated to VT1 is [18.000.000 / (200 + 400)]*200=6.000.000; to VT2 is [18.000.000 / (200 + 400)]*400=12.000.000.

Example 3: Apportion by weight
In Charge List: select charge code “KD – Quality control charge”, Apportion by “3 – Weight”
In Item List - “Size/Style/Color” tab: item VT1, input “Weight” as 2; item VT2, input “Weight” as 1.
Create Purchase Invoice with 2 items: VT1 quantity as 200 and VT2 quantity as 200. In “Charges” tab, select charge code “KD” and input amount 18.000.000. Quality control charge allocated to VT1 is [18.000.000 / (400 + 200)]*400=12.000.000; to VT2 is [18.000.000 / (400 + 200)]*200=6.000.000.

Example 4: Apportion by value
In Charge List: select charge code “BH – Insurance charge”, Apportion by “4 – Value”
Create Purchase Invoice with 2 items: VT1 quantity as 100, amount as 1.000.000 and VT2 quantity as 200, amount as 2.000.000. In “Charges” tab, select charge code “BH” and input amount 15.000.000. Insurance charge allocated to VT1 is [15.000.000 / (1.000.000 + 2.000.000)]*1.000.000=5.000.000; to VT2 is [15.000.000 / (1.000.000 + 2.000.000)]*2.000.000=10.000.000.

Voucher

Extracted from Voucher List
This is an optional field helping to limit the charge used in some vouchers. If you let this field blank, this charge can be used in all vouchers (when looking up in Charge List).
