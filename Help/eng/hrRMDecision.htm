Recruitment Decision

Entry screen

Some special notes

All applicants in the recruitment period selected in filter condition will be displayed so that you can select to input recruitment decision.
Put the cursor at the row of the required applicant, click the icon " - Recruitment Decision" to input the recruitment decision for that applicant.
Some field descriptions:

Period Code

This is period code selected in filter condition.

Applicant ID

The applicant selected to be input the recruitment decision

Employee ID

String data with maximum 16 characters; employee ID must not be one existing in “Employee Information” list.

Department

This is a required field, retrieved from “Applicant Profile Input/ Department Applied” and selected from Department list.

Job Position

This is a required field, retrieved from “Applicant Profile Input/ Position Applied” and selected from Position list.

Join Date

This is a required field, retrieved from “Applicant Profile Input/ Start Working Date” but editable.

Employee Status

This is a required field, retrieved from Employee Status list and default as “Employee Working Status” optional parameter.

Note 1:
- Applicant information will be auto-updated to “Employee Information”( except education information).
- “Recruited by”, “Recruitment Source” and “Referred by” information will be updated to “Working History/ Other”.
- “Application Form Status” (in” Applicant Profile Input”) will be auto-updated by the “Default Status of Hired Applicant”.
-“Application Form Status” in this function will also be updated.
- After the applicant is recruited, the information in “Test Result Input” and “Interview Result Input” will be updated to “Educational Background/ Recruitment Test Result”, “Educational Background/ Interview Result”.

Note 2:
When performing “Recruitment Decision”, if the applicant has already been recruited, the system will give warning message: “This applicant has already been recruited, do you want to redo the recruitment?”
- The default answer is “No”;
- If “Yes”: the ”Applicant Profile Input” screen will be displayed to edit information, except the application form status.

Note 3:
- When entering "Recruitment Decisions", if Applicant profile have attachments, the program will copy all files to Employee Information.
