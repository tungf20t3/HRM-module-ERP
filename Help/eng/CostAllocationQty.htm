Calculate Allocation Ratio by Production Quantity

In this menu, there are some calculations:

    - Calculate Allocation Ratio by Defined Ratio;
    - Calculate Allocation Ratio by BOM: used to allocate cost factor types having format as “”;
    - Calculate Allocation Ratio by Production Quantity: used to allocate cost factor types by Production Quantity formula;
    - Calculate Allocation Ratio by Cost Factor

Examples:
- Calculate Allocation Ratio by Defined Ratio: calculate only cost factors supporting to some specific departments - products;
- Calculate Allocation Ratio by BOM: calculate only material cost factors;
- Calculate Allocation Ratio by Production Quantity: calculate only cost factors with principle: more production more allocation.

Important note:
Because each type of calculation is applied only to some specific factors, they should be formatted at the first run; program will remember that format and calculate.

To test the result of this calculation, enter reports:

    - Cost Aggregation and Allocation Detailed by Raw Material;
    - Cost Allocation by Ratio;
    - Cost Allocation by Other Cost Factor.
