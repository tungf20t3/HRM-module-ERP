Physical Opening Slow Moving Item Input

This menu used to input the opening banlance of slow moving item which display in “Slow-moving Item Report” by physical stock. If the “Tracking slow moving item” parameter is “Yes”, “Last Issue date” will be also updated to next year when run “Transfer stock to next year” process. The rule of processing: base on the last issuing date in stock card data.

Last Issue Date

About the items which have arising of receiving in period but has not yet made issue, the process will update the “Last Issue date” by receiving date.
About the opening balance stock items which have no arising of issuing in period, the process will update the “Last Issue date” by “Last Issue date” of opening balance.
