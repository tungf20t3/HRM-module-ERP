Customs Declaration

Customs Declaration is used to support goods import process and may not be updated. It is used to allocate oversea expense related to goods import and calculation of import tax and VAT of imported goods.
Customs Declaration is used in 2 cases: - Non-merchant import: there is no import contract between sales - purchase.
- Merchant import: there is import contract between sales - purchase. According to these 2 cases, Customs Declaration can be created by 2 ways:

    - Directly creating: non-merchant import
    - Extracting data from import PO: merchant import

Entry screen

Field descriptions - Format view

Supplier ID

Extracted from “Supplier/Customer List”, required field.

Memo

Content, reason, purpose of making Customs Declaration

Custom

Customs House where Customs Declaration is created; extracted from Supplier List.

Dr. Term

Debit term of lot; created following Customs Declaration and extracted from “Payment Term List” to new created Customs Declaration or from declaration in import PO (optional field).

Trans. Code

Transaction code; default as 1 – Customs Declaration

Voucher No.

Number of Customs Declaration can be entered by user or automatically numbered by program depending on declaration in system

Date

Date when Customs Declaration created

Ex. Rate

Select FC Code and declare exchange rate in Customs Declaration creating date (required field)

Status

Status of Customs Declaration is automatically processed by program. There are 03 statuses: “Open”, “Complete” and “Closed”.

Field descriptions - “Details” Tab

Item Code

Code of imported item; extracted from “Item Master” to new created Customs Declaration (required field) or extracted from declaration in import PO.

UOM

Default UOM in “Item master”, can be changed by conversion of units of measurement in some special cases.

Quantity

Quantity of imported items; required field towards new created Customs Declaration or extracted from declaration in import PO.

(FC) Price

Unit price of items in import invoice in foreign currency; displayed only when currency field in format view is foreign currency. It required field towards new created Customs Declaration or extracted from declaration in import PO.

(FC) Amount

Amount of item in defined foreign currency; appears only when selected currency in format view is foreign currency.

(FC) Tax Price

Unit price to tax items in foreign currency (total unit price in import purchase invoice and expense), displayed only when currency field in format view is foreign currency.

(FC) Taxed Amt.

Amount to tax items in declared foreign currency; displayed only when currency field in format view is foreign currency.

Imp. Tax Code

Import tax code of item; extracted from “Import Tax List”

Imp. Tax Rate

Import tax rate of item

(FC) Import Tax Amt.

Amount of import tax of items in declared foreign currency; displayed only when currency field in format view is foreign currency.

Excise Tax Code

Code of excise tax of items, goods; extracted from Excise Tax List

Excise Tax Rate

Excise tax rate of goods, items

(FC) Excise Tax Amt.

Amount of excise tax calculated in defined foreign currency; this field only appears when currency field in general information is a foreign currency.

VAT Code

VAT Rate

Common operations in Customs Declaration entering

Create Customs Declaration by Extract data from Import Purchase Order

Create new Customs Declaration, click " - Extract Data from Import PO”, Input the date from which the PO is selected, then click to lookup PO. Only PO item in “Approved” status appears in the screen. Select the required PO, input the order quantity and exit the PO list screen. The selected rows in the PO list will be systematically updated to Customs Declaration.
