Hints and Tips in List Setting

List Setting

All list setting screens have common tasks, including:

New

Press “Ctrl+Insert” or click “New” toolbar icon to create new code.

Edit

Press “Ctrl + E” or click “Edit” toolbar icon to edit information of an existing code. Note: codes with data entered or transferred to accounting books are ineditable.

Save

After inputting information, click “Save” to save code.

Delete

Press “Ctrl + Delete” or click “Delete” toolbar icon to delete a code. The same as “Edit”, codes with data entered or transferred to accounting books are ineditable.

View

Put the cursor in required code, click “View” toolbar icon to view the information of the code. It is possible to add new code or edit this code.

Export

This function is used to export list information to Excel spreadsheet.

Freeze Columns

This function is used when you want to fix some columns in a multi-column list screen. Put the cursor at the column you want to fix, click “Freeze columns” icon. Then, when you use the horizontal scroll bar, the columns from the first one to the frozen one will not be scrolled or adjusted the width. To unfreeze column, select the frozen columns and click “Freeze Columns” icon again.

View list by page

The list is divided into pages numbered from 1 to n. To view list by page in order, click the number of page you want to view.

To define number of items (records) to be viewed in each page, you can select one in following default numbers: 5, 10, 15, 20, and 25.

You can limit number of records to be viewed in list screen by filter conditions.

Other List Setting

In all lists, there is some common information or definitions as follows:

Utility

In case object codes are encoded as a string in which last characters are series of numbers, when adding new code, key in one or some first characters of an object code, the system will search the highest number of the keyed string and increase 1.
To get the newest value, click the Down Arrow "↓".

For example:
The current code in Customer list is "KH004", when you key in "KH0", the system will suggest the new code as "KH005".

Other Name

Other name is used to give English description of the code; used in English reports.

Status

There are 2 statuses of code:

    -1 - Active: active codes will appear in lookup screens and used in reports.
    - 0 - Inactive: inactive codes cannot appear in lookup screens and used in reports.

Code type

Codes can consist of any characters except system master or operator characters (` ~ ! @ # $ % ^ & _ | ‘ ? “).

Rules for creating coding format

All codes of one list should have the same length and follow one format.
All codes must not be “nested”. A code is considered to be nested by another one when all characters in that code belong to ones in an existing code and vice versa.

For example: “VV001” and “VV00” are nested because 4 characters of “VV00” belong to 5 characters of “VV001”.

The reason why this rule is required is that some reports in which data are get from relative codes. If there are nested codes, data will also be get from the following codes.

For example:
There are arising in both “VV00” and “VV001”, if select to get data for stock summary report of “VV00” only, the system will also get data of “VV001”. If defining to get data from absolute code, data of “VV0%” codes cannot be get.

Add new code in lookup screens

If it is required to add a new relating code during lookup, right click on code field in lookup screen, the functional screen will appear.
The system allows add new code in following lookup screens now:

    1. Account;
    2. Customer;
    3. Item;
    4. Job;
    5. Contract;
    6. Loan contract;
    7. Expense;
    8. Province;
    9. ...

Select many items in list screens

To select many items in search screens, operate following tasks:

    - To select an item in list, tick the checkbox in front of it;
    - To deselect this item, click the ticked checkbox ;
    - To select all items on a page, click the yellow button on the top right corner, choose “Mark all on page” option;
    - To deselect all, click the yellow button again and choose “Unmark all” option

Import data from excel

In some usual lists with large number of codes as:

    - Item List;
    - Customer List;
    - ...

The system is enabled to get data from spreadsheet to save the initial data entry time.
The data getting process:

    - Enter the required list;
    - Click the icon , the system allows user to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to the list.
