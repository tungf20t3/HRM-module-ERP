Employee Workday Additional

This function used to add workdays in the special cases which you can not use menu "Employee Workday Adjusting" to update the workday.

For examples:
- Work overtime without registration;
- Work a job not belong to "Employee Working Shift Schedule";
- ...

Screen

Refer to Employee Card Info Input, Detailed Shift

Special tasks

Download Template File

Click to download template file to your computer. There are some notes during data entry in template file:
- Employee ID, department: must be ones available in lists, required fields;
- Date: input in date format, required field;
- Shift, workday: must be ones available in lists, required fields;
- Detailed shift: user-defined field;
- Time in/out: input in 00:00 format;
- In/Out next day: input 0 or 1 (check or not).

Import Data from File

Click to import timekeeping data to server;
Click “Select…” to choose timekeeping data file;
Select copy type 1 – Overwrite if you want to update all data; otherwise, if you select type 0 – No, only new rows will be inserted to the available data.

Note:
- In case of empty file, the system will not update and give message;
- If the system gives “unidentified error” message, check the number fields if they are too long or not;
- If the system indicates error in a specific cell, check the validity of the data input.
