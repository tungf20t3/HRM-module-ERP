Report Form Setup: Investment Implementation and Liquidation by Project, Work and Work Detail

There are 2 parts: Implementation and liquidation. Both 2 parts are defined with articles for construction, equipment and other expenses.
To implementation one, these items are shown in columns and contained in following columns:

    - Investment Implementation in period
    - Investment Implementation fromYear-beginning to Period-end
    - Investment Implementation from Work-start to Period-end
