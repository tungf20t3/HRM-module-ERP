Book Exchange Rate

You can recalculate book exchange rate by monthly average, moving average, FIFO methods at the end of period and create the exchange rate difference for transactions arising in period.

Field descriptions:

Period / Year

Period/ year when you want to calculate book exchange rate and create the difference transaction

Update Booked Ex. Rate again

If you check in this check-box option, the vouchers which have been calculated book rate and created difference transaction during data entry will be recalculated book rate and created other difference transaction.
If you do not check in this option, the system will calculate book rates and create difference transactions for the vouchers which have not been applied book rate by monthly average method and the difference transactions have not been created.
