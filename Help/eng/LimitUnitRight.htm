Limit Unit Access Right

Path: Modules → System → User Maintenance → Limit Unit Access Right
This function is used to change the current right of users in a group.



For illustration: A group of 20 users (I1 – I20) are assigned to unit A, but the user I5 is not allowed to edit in unit A. In this case, we can assign the rights to the whole group and then limit the edit, delete right towards user I5.
