Labor Contract and Insurance Management List

Medical Care Center List

Used for data entry in "Employee Insurance and Union Due Information" and creating reports:

    - Social, Health Insurance Declaration;
    - Employees Requesting for SI Book, HI Card

SI Benefit Item List

Used for data entry in:

    Data Entry for Social Insurance Benefits;
    Data Adjustment for Social Insurance Benefits.

And creating "List of Employees with Social Insurance Benefits" report

SI Benefit Item Type List

Used to define benefit in "SI Benefit Item List. The category (1 - Sickness, 2 - Maternity, 3 - Convalescence, 9 - Other) is base to calculate data for "List of Employees with Social Insurance Benefits" report

SI Change Kind List

The category (1 - Increase, 2 - Decrease, 3 - Change) is used to define type of changes of SI information in "SI Change Type List", is base to calculate data for "Employees Joining Insurance" report.

SI Change Type List

Used to define SI information which is base to calculate and create "Employees Joining Insurance" report.

Insurance and Union Due Information

Used to define information for calculating B01(Social Insurance), B02(Health Insurance), B03(Unemployment Insurance) and C01(Union Due) in salary and base for generating data for "Employees Joining Insurance" report.

Insurance Contribution Rate Ceiling

Used to define information for calculating and creating "Employees Joining Insurance" report.

Contract Type

Used as filter condition in contract reports

Minimum Wage Levels

Used to define the minimum wage being base to calculate pay type codes:

    - L00 - Actual salary;
    - L01 - Salary by Working Hour;
    - B00 - Base Salary;

and being base to create reports:

    - List of Employees with Social Insurance Benefits (Maternity);
    - List of Employees Joining Insurance.
