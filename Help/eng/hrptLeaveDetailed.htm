Employee Leave Detailed Report

Filter Condition

Descriptions

The data in this report can be generated after workday computation from information in Employee Leave Input with the workday sign as one defined in "Annual Leave Sign" parameter; and viewed by date from/to in filter condition.

In case the priority in Employee Leave Input is 2 - By non-working day or public holiday, if there is a day of leave on the non-working day or public holiday, that day will be neither subtracted on annual leave nor viewed in this report.
