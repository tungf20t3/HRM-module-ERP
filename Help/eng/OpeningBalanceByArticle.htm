Input Norm Opening Balance

This function is used to input opening balance by norm (if any) of projects/jobs before using the program. Opening date is the beginning date of data entry defined in “Declare Beginning Input Date”. Opening year balance (if any) will be input in account in General Ledger.
Users have to input opening balance only once when beginning to use program. Balance of next periods and of next years will be automatically calculated and transferred by program.
Path: Module → Investment Accounting → Data Entry → Input Norm Opening Balance
