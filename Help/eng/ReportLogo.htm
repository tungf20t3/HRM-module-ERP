Set Logo for Report

This utility is used when you want to print Sales Order and Requisition with the logo on it.
You can create many logos, then edit the logo name or select another file to have new logo. You can also delete the old logo if you don’t want to use it any more.
To have a logo printed in a voucher or report, you have to select the logo and click icon "Change the defaul logo" to activate the logo.
