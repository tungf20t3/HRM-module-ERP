Slow-moving Item Report

Property This report shows quantity, value of items which are slow to move in defined number of days (days items hasn't been issued). Slow moving items must be in stock; last issue date must be sooner than report date and period between report date and last issue date mustn’t be greater than defined number of days. Note: transferred items are not slow moving ones.

"Slow-moving" concept in this version simply describes how many days the stocks have not been moved in or out and their quantity.

Note:
Stocks in transfer are not considered as slow-moving items.

Report layout

Field descriptions:

To date

Reporting date

Number of days

Number of days in which items mustn’t be issued before reporting date
