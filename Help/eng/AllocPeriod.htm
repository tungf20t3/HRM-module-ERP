Definition of Periodic Allocation Entries

It’s different from allocation transaction, periodic allocation transaction is used to allocate available amount periodically. (It can be considered as fixed asset or tool & supply depreciation allocation). These entries can be used to allocate:

    - Goodwill when purchasing shares of subsidiaries
    - Long-term advanced expenses;
    - Unrealized revenue.

Entry screen

Field descriptions

Unit

Default to be current unit

Entry code

A required field

Allocation Type

Default to be 1; only 1/2/3 numbers are allowed.

Allocation Period(s)

Calculated by month and be default as 0 but editable.

Start Date

A required field

End Date

System-generated by start date plus number of allocation periods; shown as the last day of the month but editable

Debit/Credit Account

Required fields with the last sub-accounts

Currency

Required field and default as currency of the current record but editable

Customer

Required field in case debit or credit accounts are AR-AP accounts

Other fields

Optional fields for different businesses
