Item Type List

Path: Modules → Inventory Management → List Maintenance → Item Group List


Item type list is used to categorize items into 5 types: 21- Raw materials, 31- Tools & supplies, 41- Semi-finished products, 51- Finished products, 61- Goods. This also supports stock management and reporting by item type.
Item categorizing helps in special functions which will be developed during application deploying. Item type 51- “Finished goods” is now used to define costing objects in cost calculation.
Item type list is fixed in system setup, only item type description is editable.

See “Hints and Tips in List Setting” for details of common tasks in list maintenance
