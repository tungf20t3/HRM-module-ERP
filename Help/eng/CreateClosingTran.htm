Closing Entry

Path: General → Closing Transaction → Closing Entry
This function is used to system-generate closing entry based on the entries defined in “Closing Entry Definition” menu.

Closing entry screen

Field descriptions

Select

Check-box option to select the entries to be closed in period

No./ Description/ Closing Type/ Debit Account/ Credit Account

See “Closing Entry Definition” guide for more details

Some basic tasks

Select/ Unselect

Click on “Select” check-box to select or unselect the entry

Select all

Click the arrow -above to select all the entries in list to be closed

Unselect all

Click the arrow again to unselect all selected entries in list

Create Closing Entry

Click “ - Create closing entry” to system-generate entries in ledger

Delete Closing Entry

To delete the generated entries in ledger, select the desired entries, click “ - Delete closing entry”.
