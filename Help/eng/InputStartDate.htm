Declare the beginning input date

Path: Modules → System → Declare the beginning input date

Declaration screen



When starting use AI ERP, it is required to input the following balances to continue the information management:

    - AR-AP balance detailed by customer/supplier
    - AR-AP balance detailed by customer/supplier and invoice (if any)
    - Inventory balance
    - Other account balance

If the beginning input date is difference from the start date of fiscal year, you have to input both account balance and year-opening balance to get the balance sheet for the first year using AI ERP.
