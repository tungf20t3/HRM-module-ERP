Income Defining by Pay Type

This function is used to define the kind of taxable income for the calculated pay types.

For example: To piece rate salary
To resident individuals with labor contract in term of 3 month or over, define the "Progressive Tax Tariff" field as "A01.";
To resident individuals without labor contract or with under-3-month contract, define the "Flat Rate Tariff" field as "A02.";
To non-resident individuals, define the "Non-resident" field as "B01.".

Other taxable incomes (not being salary) will be input in "Income/Deduction by the Progressive Tax Tariff" or "Income by the Flat Rate Tariff" functions.

Pay Type Code

Selected from "Pay Type" definition

Allowance

In salary, allowances are sum in one code. In PIT computation, it is required to define which allowance are taxable income.

Progressive Tax Tariff

Selected from "Income/Deduction List" which are used for resident individuals with labor contract in term of 3 month or over.

Flat Rate Tariff

Selected from "Income/Deduction List" which are used for resident individuals without labor contract or with under-3-month contract.

Non-resident

Selected from "Income/Deduction List" which are used for non-resident individuals.

Type

Used in case of salary in foreign currency. If 1 - Income, this type code will be calculated by salary exchange rate; if 2 - Other, it will be calculated by insurance exchange rate.
