Access Right Assignment

Path: Modules → System → User Maintenance → Access Right Assignment
This function is used for system administrator to assign the access right to other users. Enter Access Right Assignment function, select the required user and click edit button.

To assign, select each module branch and check in the boxes before. If a module branch is selected, all menus branches inside are also selected. However, it is possible to select each small menu and assign by task ("New", "Owner's Edit", "Owner's Delete", "Full Read", "Full Edit", "Full Delete", "Release Later", "Release Now",...). When a common user (not in admin group) performs “Edit” or “Delete” vouchers created by other users, the system will check 2 rights "Full Read" (View Vouchers Created by Other Users) and "Full Delete"(Delete Vouchers Created by Other Users).
