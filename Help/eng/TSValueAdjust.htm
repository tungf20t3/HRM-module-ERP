Tool & Supply Value Adjustment

If you want to adjust allocation and increase value, use function “T&S Value Adjustment”
This function is the same as creating a new tool & supply but the difference is that allocation will be grouped to the adjusted tool & supply.
The operations in this section are the same as ones in List Maintenance entry.
Path: Modules → Tools & Supplies Management → Data Entry → Tool & Supply Value Adjustment

Entry screen


Field Descriptions

Tool & Supply Code

Code of adjusted tool & supply, extracted from “Tool & Supply List”

Period / Year

Month/year of adjustment

VC No. / Date

Number of voucher related to this adjustment

Capital Code / Reason

Sourced capital and reason of original cost increase; capital code is extracted from Capital List and reason code is extracted from increase /decrease reason list.

Currency Code

Currency code is extracted from tool & supply list.

Ex. Rate

Exchange rate is extracted from tool & supply list; editable.

Original cost / Accumulated Allocation / Current Value

The information relating to tool & supply increase value adjustment. Add foreign currency field for tools & supplies having foreign currency value.

Remaining Periods

Remaining allocation periods of tool & supply after adjustment; system-generated but editable. If tool & supply adjustment makes the allocation time changed, you have to enter it. This field is an important base to redefine monthly allocation value after adjustment.
This information is valid to tool & supply allocated by line method.

Monthly Allocation Adjustment

Monthly allocation value is automatically calculated following current allocation regulations.
This information is valid to tool & supply allocated by line method.

Allocation Value

This field is system-calculated and read-only. It shows periodic allocation value of the tool and supply after adjustment and the allocation will be calculated based on this value.
If you want to change the value, you can adjust the value at “Monthly Allocation Adjustment” field only.
This information is valid to tool & supply allocated by line method
