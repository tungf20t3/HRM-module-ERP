Income by the Flat Rate Tariff

Field descriptions – Format view

Pattern Code (Income Code)

Required field; extracted from Income/Deduction List (only income code having “Income Type” with “PIT Type” as Entire -> defined in PIT Assessable Income List).

Field descriptions – Grid view

Taxable Income

Manually input Available to input only when income code with “Nature”=1 – Income (defined in Income/Deduction List)

Deduction Amount

Default value in accordance with “Income Code” (with value of “Tax Threshold” defined in Income/Deduction List), editable.

Assessable Income

System-generated; equals taxable income subtracting deduction amount. Assessable income will be 0 if the result is less than 0 or the “Tax Assessment” option in Employee Information is 0 - No.

Tax Amount

System-generated; equals assessable income multiplied by tax rate (%)
