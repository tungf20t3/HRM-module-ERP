Moving Average Exchange Rate

This function is used to calculate booked exchange rate by day and create exchange rate difference of transactions in period.

Field descriptions

Period/Year

Period/ year when you want to calculate booked rate and create exchange rate difference transaction

Currency

Foreign currency that you want to define book rate

Unit

You can select the assigned unit (with New, Edit or Delete right) to calculate booked rate. If this field is blank, the exchange rate will be calculated for all assigned units.

Calculation Type

There are 2 options:

    - Calculate Moving Average Exchange Rate: for accounts with foreign currency;
    - Delete Moving Average Exchange Rate Transaction: delete the exchange rate and related transactions.
