PR Approval

This is the next step after creating Purchase Requisition. If company has purchase process from Purchase Requisition, there must be PR Approval; otherwise, Purchase Order can not be created from data in PR.
There are 2 ways to implement PR Approval:

    - Directly approve in creating PR screen
    - Approve in function “PR Approval” screen

To approve PR which has not been transferred to PO, there is “Undo” function to cancel the approval.

Note that user must have right to approve correlative vouchers; approval right is declared in “Approving Officers”.

PR Approval Process

    Previous step: Creating Purchase Requisition
    Next step: Supplier Allocation (Creating Purchase Order)
    Created Dept.: Approve Department

Note:
If in a PR with many items, some are approved and the others (approved quantity=0), user can not apply “directly approve in PR” method but use function in “Approval PR” screen.

Approval screen

Some operations in Purchase Requisition Approval

Approve PR in “Open” status

Open “PR Approval” function, click on the “Select” check-box in required rows, then click " - Approve”. The approved PRs will disappear in this screen and changed to “Approved” status.

Detail approving the Purchase Requisition in “Open” status

To approve a part of Purchase Requisition, open “PR Approval” function, put the cursor at the required row, then click “ - Detail Approve”. The PR will be opened as voucher entry screen. Check "Approved" check box on the detail rows to be approved and save. The selected rows will be transferred to “Approved” status.
If you do not accept PR row, Check "Cancelled" check box on the detail rows to be approved and save. The selected rows will be transferred to “Opened” status.

Cancel the Approval of “Approved” PR

Open “PR Approval” function, click on " - “Search” icon, select “2 - Approved” status in “Other” tab, click “Ok”. The approved PRs will be listed. Tick on the “Select” check-box in required rows, then click “ - Cancel”. The selected PR will disappear in this screen and changed to “Open” status.

Cancel PR

Open “PR Approval” function, tick on the “Select” check-box in required rows, then click “ - Cancel”. The cancelled PR will disappear in this screen and changed to “Closed” status.
