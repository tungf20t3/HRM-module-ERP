Input Norm Planning Amount

To support planning comparison reports, user must input project/job planning amount in a specific screen at the beginning of each year.
Path: Module → Investment Accounting → Data Entry → Input Norm Planning Amount

Entry screen

Field descriptions

Year

Yearly plan of project/job having planning amount

Project

Code of project/job having yearly planning amount

Norm

Report norm of project/job having yearly planning amount

Amount/FC Amount

Amount to implement yearly plan of project/job
Reports:

    - Investment Implementation by Project, Work, Work Detail;
    - Investment Implementation;
    - Other Expenses Report
    - PMU Expenses.
