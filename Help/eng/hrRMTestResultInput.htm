Test Result Input

Entry screen

Format view – Upper area

Applicant

Only applicants in the current recruitment period

Recruitment Round/ Content

Recruitment round and content will be selected from examination forms which have been input in recruitment period information.

Note:
The examination form will be selected following the defined “Recruitment Period – Required Position – Requesting Department”. If there is no information about “recruitment period – required position – requesting department”, the system will select all.

Date of Test/ Score

This information will be displayed in “Test and Interview Result” report.

Grid view

Test Type

Test type can be manually input or retrieved from test requisition.
After inputting details, if you want to edit the general information of recruitment round, you have to update the details manually. In case of retrieving date, the system will not delete the old test types.

Score/ Result

This information will be displayed in “Test and Interview Result” report.

Format view – lower area

Result

There are 2 options: 1- Passed and 2 – Failed.

Status

Application form status will be updated on the basis of the “Result” and information in “Examination Form Input”, editable.

Note:
The application form status will be auto-updated to “Applicant Profile Input” after there is the test and interview result:
- The status will be updated following one of the greatest recruitment round.
- In case:
o All test results are deleted, the application form status will be updated following the default one in optional parameter.
o All interview results are deleted, the application form status will be updated following one of the test result of the greatest recruitment round.

Retrieve data from test requisition

“Test Type”, “Test Content” can be got from examination form which has been defined in recruitment period. Click the “– Retrieve data from test requisition” icon, the system will get test types from test requisition to grid.
