Production Overhead Allocation Transaction

This function is used to allocate production costs accounts to WIP account in case WIP account is split into many sub-accounts which receive the production costs.
Path: Modules → Advanced Costing → Production Overhead Allocation Transaction

Note:
In case WIP account is not split into sub-accounts, you can use “Post Transaction” function in General Ledger module instead of this one.
Only cost factors with “Auto Post to WIP Account” in Cost Factor List are used in this function.
