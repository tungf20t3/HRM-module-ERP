VAT Declaration (Circular No. 18)

Filter condition screen

Article definition screen

Field descriptions

Calculating Way

There are 3 options to select the way to calculate value of each article in report:

    - 0 – Base on Formula: there is 1 formula for each article, this formula includes addition and subtraction only.
    - 1 – On Opening Balance: debit, credit accounts can be user-defined. According to user-defined options, the system will get only debit or credit balance and balance will be shown in which column: 1 – Tax, 2 – Revenue or 0 – Both.
    - 2 – On Acct. Arising: debit, credit accounts can be user-defined. According to user-defined options, the system will only get the arising amount in debit or credit arising, or in debit – credit corresponding.

From File

According to tax account type, data can be selected to get from: 0 - GL, 1 – Input VAT or 2 – Output VAT.

Tax Code

This information helps the system to define by which tax code the data will be get for the article.

Input VAT

If data are defined to get from input VAT vouchers, there are 2 options: 0 - All, 1 – Voucher with Tax only. In some cases, there is purchase revenue but tax value. If only revenue from vouchers with tax can be get, select option 1.

Debit Accounts

Define the accounts of which debit arising is used. This is valid when calculating way is "2 – On Account Arising"

Credit Accounts

Define the accounts of which credit arising is used. This is valid when calculating way is "2 – On Account Arising"

Formula

Formula to calculate this article value from the other articles in the declaration

View Type

In VAT declaration, there are 2 columns revenue and tax. Therefore, it is required to define which data can be get for each article: revenue, tax or both.

Note:
Input, output VAT adjustment articles are not declared in this report but entered in adjustment reports, “VAT Declaration” one.
