Personal Income Tax Finalization

It’s required to have the “Personal Income Tax Finalization” function processed so that the system can aggregate arising data in year before using annual reports. Filter condition:

    - Year: required field
    - Unit: personal income tax finalization can be processed separately by each unit
    - Replace Income Type: 0 – No, 1 – Yes. Default as 1 in the first running time, next times depending on the latest one.

The calculation result will be viewed in reports by year (annual)

"Replace Income Type" is used in case the taxpayer is an employee with and without contract; resident and non-resident individual in the year of PIT finalization. If the "Replace Income Type" is 1 - Yes, the "Income Type" will be replaced.

Note:
- The replacement is only applied for employees with "Finalization Authorization";
- If there is no input for replacement, the current income is remained.
