Tool & Supply Opening Quantity Input

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:

    - Enter Tool & Supply Opening Quantity;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file.

Notes of inputting data in template file:
- Expiry Date: input in date format;
- Employee, Tool & Supply: must be one existing in the corresponding list (Employee, Tool and Supply List) with the same length in add (edit) screen in list; employee ID must be one belonging to departments in the assigned units;
- Quantity: input in number format, greater than 0.

The system will give error message in some following cases:
- Data in column containing employee, tool and supply do not exist in the corresponding lists. In this case, the system will indicate the specific invalid cell;
- String data is longer than the available length of field (indefinite error);
- Wrong date format;
- Quantity is less than or equal to 0.

Other notes:
- You must have the "New" right to import new data to voucher detail grid;
- You must have the "Edit" right to edit data in voucher detail grid.
