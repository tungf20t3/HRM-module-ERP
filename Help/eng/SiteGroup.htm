Warehouse/ Site Group List

A business, in fact, has many warehouses/sites with different purposes. For example, site of goods for whole sales, retail or sales promotion…. If the common average cost of all sites is calculated, the cost of goods sold will not be correct.

Therefore, warehouse/site groups are defined to group warehouse/site with the same purpose; supporting in cost calculating by warehouse/site group.
For more details, please refer to:

    - Optional parameters
    - Cost calculating functions (average cost, FIFO cost)
