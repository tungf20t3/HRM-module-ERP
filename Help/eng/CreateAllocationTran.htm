Allocation Entry

This function is used to system-generate allocation entry based on the entries defined in “Allocation Entry Definition” menu.

Note:
If the parameter option “System-generate allocation coefficient detailed by unit”” is 1 – Yes, the “Unit” field in filter condition of this allocation transaction will be required and selected from list of units that you have the “New” right only;
All tasks in allocation transaction (edit, allocate, delete, calculate) must be by selected “Unit”.
The detailed allocation coefficient will be save by unit.

Allocation entry screen

Field descriptions

Select

Check-box option to select the entries to be allocated in period

No./ Description

Ordinal number and description of the allocation entries in accordance with ones defined in “Allocation Entry Definition” menu.
See “Allocation Entry Definition” guide for more details

Account

Number of source account in allocation entry. Although this account is defined as general or subsidiary account, when running allocation, the system will only allocate from subsidiary accounts of a general account.

Allocation Type

If allocation type is Dr. Account to Cr. Account, the double entry of allocation will be Debit on account receiving allocation (Target Account) and Credit on allocated account (Source account).
If allocation type is Cr. Account to Dr. Account, the double entry of allocation will be Debit on source account and Credit on target account.

Some basic tasks:

Calculate allocation coefficient by account

This function is used in case allocation coefficient is not input by month but system-extracted from the value of coefficient account.
Coefficient account is defined in “Allocation entry definition” menu. See “Allocation entry definition” guides – debit and credit account – for more details.
To calculate allocation coefficient by account, click “

Calculate allocation coefficient by account”. Before calculating, you have to select the entries required to be calculated allocation coefficient by account. Otherwise, the system will give message: “No entry selected…”.

Edit allocation coefficient

If have no coefficient formular in detail,we may use “Edit allocation coefficient” function.
Select the desired entry, click “Edit allocation coefficient” or press Ctrl + E → “Edit coefficient” screen appears. Key in the desired coefficient and click “Save” to save or “Cancel” to cancel the new coefficient.

“Edit allocation coefficient” screen

Create allocation entry

After defining allocation coefficient and amount, to create allocation entries in ledger, select the desired entries, click “Create allocation entry”.

Delete allocation entry

To delete the created allocation entries in ledger, select the desired entries, click “ Delete allocation entry”.

Reset allocation period

To select another period in allocation screen, click "Select Period".
