Input Job/Project Accumulation byJob Norm

Besides inputting opening balance of project/job, user must input accumulative amount from starting work. User can input it once when beginning to use the program.
Path: Module → Investment Accounting --> Data Entry → Input Job/Project Accumulation by Job Norm

Entry screen


Field descriptions

Project

Code of project/job having accumulative amount

Norm

Report norm of project/job having accumulative amount

Dr. Accumulate / FC Dr. Accumulate

(Foreign currency) debit accumulative amount from starting work to the beginning of year.
Reports:

    - Investment Source;
    - Detail of Investment Source;
    - Investment Implementation by Project, Work and Work Detail;
    - Investment Liquidation by Project, Work and Work Detail;
    - Investment Implementation;
    - Other Expenses Journal;
    - PMU Expenses Journal.

Cr. Accumulate / FC Cr. Accumulate

(Foreign currency) credit accumulative amount from starting work to the beginning of year

Opening Dr Accumulate / FC Opening Dr Accumulate

(Foreign currency) debit accumulative amount from the beginning of year to when you start inputing data.
Reports:

    - Investment Source;
    - Detail of Investment Source;
    - Investment Implementation by Project, Work and Work Detail;
    - Investment Liquidation by Project, Work and Work Detail;
    - Investment Implementation;
    - Other Expenses Journal;
    - PMU Expenses Journal.

Opening Cr Accumulate / FC Opening Cr Accumulate

(Foreign currency) credit accumulative amount from the beginning of year to when you start inputing data
