Tool & Supply Group List

Path: Modules → Tools & Supplies Management → List Maintenance → Tool & Supply Group List
Tool & supply group list is used to classify tools & supplies in groups, used in tool & supply reports by group.
The list is designed with 3 group types. A group type consists of many tool & supply groups.

The group code is unique in a group type (not duplicate or overlap); however, it can be used again in another group.

See “Hints and Tips in List Setting” for details of common tasks in list maintenance
