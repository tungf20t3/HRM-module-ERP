PR Close

Common tasks in closing PR

Close PR

In PR browser, check on PR to close, then click on - “Close” to close PR.

Undo PR Close

In some cases, you are required to undo “Close” of a PR to operate some previous steps.
Click “ - “Search” to filter closed PR. All PR in “Close” status meeting filter condition are listed.
Check on PR to undo close
Click on - “Roll Back” to back status before closed.
