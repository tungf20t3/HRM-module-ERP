Voucher Book Using Definition

Path: Modules → System → Voucher Book Definition
It is required to define the voucher book to use before using.

Field descriptions

Voucher

This field is used to define which voucher can use voucher book. Voucher is extracted from voucher list. A voucher book can be used for many vouchers.

Book

Voucher book is extracted from voucher book list and appears in voucher entry using voucher book.

User(s)

Users authorized to use this voucher book. Many users can use one voucher book. Check to select available users from list.

Apply
In General Voucher entry defined to use voucher book, click on the search icon in Voucher Number field to select voucher book.
The system will auto-generate the voucher number to the number end defined in voucher book list and give message “Voucher book has been used all” if the last number has been used.
(See "Voucher Book List" )
