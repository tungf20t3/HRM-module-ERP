Pick List

Pick List is used in case enterprise sells by issuing first and invoicing after that or sell to non-residents. In this case, Pick Lists are created before Sales Invoices.

Field descriptions - Format view

Cust. ID

Customer ID; extracted from Customer List and is a required field.

Recipient

Representative of customer who receives goods, freely entered

Trans. Code

There are 2 options. Select 2- “Issuing Order” if it is required to have issuing order before pick list. The default code is 1 – “Pick List”.

Memo

Reason of issuing goods

Voucher No.

Number of Pick List; automatically numbered by program following declaration in voucher list system or entered by user.

Date

Date when Pick List is created

Status

There are 03 statuses of Pick List: “No action”, “Open” and “Issue”

Field descriptions - “Details” Tab

Item Code

Item code in Pick List; directly entered in “Item Master” screen or extracted from previous steps such as “Sales Invoice” or “Sales Order”.

UOM

Units of measurement of issuing items; extracted from Item Master.

Site ID

Site issuing items in Pick List; directly entered in “Site List” screen or extracted from previous steps such as “Sales Invoice” or “Sales Order”.

Location

Location of items issued in site; extracted from “Location List”, used in case that enterprise controls items by location.

Lot

Lot number of goods, used in case you wish to control items by lot. Lot declaration has been done in Inventory Management.

Quantity

Quantity of goods in invoice; directly entered or extracted from Quantity field in SO / Pick List.

Job

Job code; used to control a sale process.

SO No./ SO Line/ Inv. No./ Inv. Line

Sales Order number, Sales Invoice number and line of items in SO, Sales Invoice (if data are extracted from SO or Invoice)

Invoice Qty

Invoiced quantity of goods in SO

Delivery Qty

Real quantity of items which have been delivered to customer

Job

Code of job; used to control a sale process

Field descriptions - “Other Information” Tab

Issuing Order No./ Date

In case data in Pick List are extracted from Issuing Order, these fields will be automatically displayed. Issuing Order number and date are used for reference.

Some common tasks in creating Pick List or Issuing Order

Create Pick List - Extract from Issuing Order (SO→ Issuing Order→ Pick List→ Sales Invoice)

Create a new Pick List, select transaction code 1, click on function “ - Extract Data", select “Extract from Issuing Order” function and type start date. Lookup and choose Issuing Orders in Issuing Order selecting screen.
Input invoice quantity then exit Issuing Order screen. Data in selected rows in this list will be system-updated to Pick List.

Create Pick List - Extract from SO (SO→ Pick List→ Sales Invoice)

Create a new Pick List, select transaction code 1, click on function “ - Extract Data", select “Extract from SO” function and type start date. Lookup and choose Sales Orders in “Approved” or “Issuing” status in “SO Select” screen.
Input invoice quantity then exit Sales Order List screen. Data in selected rows in this list will be system-updated to Pick List.

Create Pick List - Extract from Invoice (SO→ Sales Invoice→ Issuing Order→ Pick List)

Create a new Pick List, select transaction code 1, click on function “ - Extract Data", select “Extract from Invoice” function and type start date. Lookup and choose Invoices in “Release Later” and “Posted to GL” status in Invoice selecting screen. Input invoice quantity then exit Invoice List screen. Data in selected rows in this list will be system-updated to Pick List.

Create Issuing Order - Extract data from SO

Create a new Pick List, select transaction code 2,click on function “ - Extract Data", select “Extract from SO” function and type start date. Lookup and choose Sales Orders in “Approved” or “Issuing” status in SO selecting screen.
Input invoice quantity then exit SO screen. Data in selected rows in this list will be system-updated to Pick List.

Create Issuing Order - Extract data from Invoice (First Invoicing process)

Create a new Pick List, select transaction code 2, click on function “ - Extract Data", select “Extract from Invoice” function and type start date. Lookup and choose Invoices in “Release Later” and “Posted to GL” status appear in Invoice selecting screen.
Input invoice quantity then exit Invoice screen. Data in selected rows in this list will be system-updated to Pick List.
