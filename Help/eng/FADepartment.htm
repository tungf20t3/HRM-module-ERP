Fixed Asset Using Department List

Path: Modules → Fixed Assets Management → List Maintenance → Fixed Asset Using Department List
This menu is used to define the departments using fixed assets. The list is used in fixed asset information entries, asset transfers and asset reports by department.


Field descriptions

Department Code

This code determines by which department the asset is managed. It is used in asset situation reports only.

Expense Department

This code determines to which department the asset depreciation expense is calculated. The expense department is extracted from Department List in System module.
In fixed asset depreciation allocation entry, the expense department is attached to account saved in accounting book.
