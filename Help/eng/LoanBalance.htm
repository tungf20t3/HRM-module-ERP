Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.

The data importing process:

- Enter "Loan Contract Opening Balance" menu;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to opening balance file in the system.

Notes of inputting data in template file:
- Unit, Account: Unit is one of the assigned units; account must be general account and AR/AP account. The input unit, account code must be one existing in the list with the same length in add (edit) screen in list;
- Debit Balance, Credit Balance: number format, 14 digits and 2 decimal digits maximum (standard format);
- FC Debit Balance, FC Credit Balance: number format, 12 digits and 2 decimal digits maximum (standard format).

Other notes:
- Cannot import data after system or unit data closing (the data closing date after the beginning input date);
- The current user must have the "Edit" right in this function to update the available data in the system (Copy type 1 - Overwrite);
- The current user must have the "New" right in this function to add new data to the system (Copy type 1 - No);
- The current user must have the "Edit" right in "Access Right by Unit" function to update the available data in the system (Copy type 1 - Overwrite);
- The current user must have the "New" right in "Access Right by Unit" function to add new data to the system (Copy type 0 - No);
- Enable to import data for many units. If the data of the current unit has been closed, the system will give warning message when you click the icon;
- If data of other units has been closed, the system will also give warning message during importing process and indicate the specific invalid cell;
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
