Services Sales Return Invoice

This voucher is used to record services sales return transactions.

Field descriptions – Format view and Other information

Refer to “Sales Return” for reference

Field description – Grid view

Debit Account

Code of return account

Amount

Returned amount in transaction currency

Tax Code/ Tax rate /Tax Account

Code of tax rate. If this field is blank, “Tax rate”, “Tax Account” and “Tax” fields will not be generated. Tax rate appears after tax code is entered; auto-displayed in accordance with tax code. Account of deducted output VAT; default to appear according to tax code

Tax

Tax amount in transaction currency. This value is equal to amount multiplied by tax rate, but editable.
This voucher is used for output VAT report with negative value.

Tax Authority

See “Field descriptions” of grid view in “Sales Invoice” for details

Description

Description about transaction
