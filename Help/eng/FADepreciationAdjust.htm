Adjust Monthly Depreciation

If you want to change depreciation of a month but do not want to change ones of next months, use “Adjust monthly depreciation” function.

Entry screen

Field Descriptions

FA Code

Code of adjusted asset, extracted from Asset List

Period

Month of depreciation adjustment

Edit

Select “1 - Edit" to input adjusted value

FC Code

Currency code is extracted from fixed asset information list.

Ex. Rate

Exchange rate is extracted from fixed asset information list by each FA Code.

Original cost / Accumulated Depr. / Current Value

Reference information, mustn’t be edited Add foreign currency fields for assets having foreign currency value

Monthly Depr.

Input adjusted value, can only be input if edit field is “1”
