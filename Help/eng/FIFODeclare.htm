Warehouse information for FIFO cost calculating

Path: Modules → Inventory Management → List maintenance → Warehouse information for FIFO cost calculating.



Warehouse information for FIFO cost calculating screen

Appear site in site list and allow to correct site series. smaller site is preferential to calculate before FIFO cost calculating. This function use to edit the order of site in FIFO calculating algorithm, that site with higher order will be specify to process first in case of circular–stock inter-transference.

Example : transfer item VTA from A site to B site, then from B site to sale at the same day.
B Site cost of goods sold calculating is firstly calculated A site cost so warehouse information for FIFO cost calculating will define that A site series is smaller than B site. To know site B Site cost of goods sold, that calculate A firstly is necessary, so that, in “Warehouse information for FIFO cost calculating”, we should setup the order of site A higher than B.
