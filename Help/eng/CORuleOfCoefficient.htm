Setup Rule of Coefficient

In menu “Set up Coefficient of Distribution”, we have defined distribution coefficients. In general case, costs are distributed to all aggregating objects by that coefficient.
In special case, if there are some objects receiving that cost, it is difficult to carry out the distribution by coefficient in menu “Set up Coefficient of Distribution”. At that time, we should use menu “Set up Rule of Coefficient” to limit.

Example:
There are 10 workshops producing 1 same product, only 1 workshop has not spent finished goods stored expense because there is local site there. All other 9 workshops must spend these expenses. There are 2 ways to enter data:
- Detail cost vouchers of 9 workshops and monthly post in general ledger as these details;
- Only entry for 1 total number of all workshops, then declare once for other 9 workshops in menu “set up Rule of Coefficient”.

Before, it takes long time to separately define a cost factor for each product. The system now supports to do it quickly by “New rule of coefficient for multi-factor and multi-product” function.
This function helps to define mass of cost factors for many products. The system will base on the combination of information defined in entry screen – usually cost factor and product – to create mass of allocation details for product.

Entry Screen

Field Descriptions

Period / Year

Period / Year are closing time, defined by order from the beginning of fiscal year.

Factor Code

Code of cost factor having coefficient of distribution

Indirect Dept.

It is used as intermediary section to distribute to other sections.

Department

Code of production line, workshop defined to receive distributed cost

MO Number

Number of MO defined to receive distributed cost

Product

Code of product defined to receive distributed cost

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.

The data importing process:

- Enter "Setup Rule of Coefficient" menu;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file into the system.

Notes of inputting data in template file:
- Factor Code, Product, Indirect Department, Department, MO Number: must be one existing in the corresponding lists (Cost Factor List, Item List, Section List, Manufacturing Order) with the same length in add (edit) screen in list;
- Indirect Department: must be one existing in Section List, which is not checked in "Direct" option;
- Department, MO Number: are optional fields depending on structure of the costing problem;
- Product: user-defined information, 128 characters maximum.

Other notes:
- Cannot import data after system data closing date (the data closing date after the beginning input date);
- The current user must have the "Edit" right in this function to update the available data in the system (Copy type 1 - Overwrite);
- The current user must have the "New" right in this function to add new data to the system (Copy type 1 - No);
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
