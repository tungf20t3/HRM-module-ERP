View the Approved Annual Leave Request

Used to view the annual leave request approved via SMS.

Send leave request via SMS

Syntax

L[Space]number of days[Space]d.M.y
- M and y can be blank, it is default as the current month/year;
- M can be with or without 0 before (e.g. 2 or 02);
- y must be 4 digits if input;
- If number of days is blank, it is considered as 1 day.

Note: the space is accepted with 2 or 3 characters (e.g. L˽˽5˽˽˽01/02/2020)

Data Process

- When receiving message, the system will base to the declared phone number to determine the employee sending message;
- The system will check and give message to the employee is following cases:
+ If the employee has a pending leave request, the message will be "You have already had a leave request.";
+ If the number of days requested for leave is over the remaining days of annual leave, the message will be The number of days requested for leave is invalid.;
- If the request passes all checks, the system will send this request to supervisor of the employee, for example:

"Nguyen Van A requests to leave in 2 days from 21 Jan 2020.
The annual leave balance of Nguyen Van A is 3 days.
To approve the request, send XXXXXX, to reject the request send XXXXXX."

- The system will also send the employee a inform message: "Your request has been sent to your supervisor for approval".
- After receiving message from the supervisor, the system will send the employee the answer for the request and send the supervisor a inform message.

The request message is valid if the request date is after the current date;
After n days, the code to approve or reject will be no longer valid.

In case of wrong syntax or invalid code, the system will send the sender the message: "Your message is invalid."
