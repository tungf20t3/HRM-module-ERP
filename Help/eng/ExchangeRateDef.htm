Exchange Rate Difference Entry Definition

Path: Modules → General Ledger → Definition → Exchange Rate Difference Entry Definition
Before processing the year-end revaluation of foreign currency accounts, we have to define the exchange rate difference entries towards required accounts.

Entry screen



Field descriptions

Ordinal Number

Ordinal number of exchange rate difference entries by which these entries will be processed.

Description

Description of exchange rate difference entry. This is required field and will be used as description in ledger of accounts defined in the exchange rate difference entry.

Account

Account of which the exchange rate will be revalued at period-end; extracted from “Chart of Accounts” with the defined currency different from base currency. The fact that the account is recorded in debit or credit depends on the result of revaluation (loss or gain).

Exchange Rate Difference Account

Account used to record the difference amount at period-end; extracted from “Chart of Accounts”. The fact that the account is recorded in debit or credit depends on the result of revaluation (loss or gain).
