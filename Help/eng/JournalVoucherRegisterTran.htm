Journal Voucher Register

You must register journal voucher before viewing journal voucher books.
The journal voucher creating process:

Add new journal voucher

Follow the following steps to create new voucher:

    - Open the function “Journal Voucher Register”, don’t choose “Copy Journal Voucher Register To Next Period”. Press Ctrl+Ins (in case there are some journal vouchers available), or click on the icon “New”;
    - Input filter conditions to group journal voucher (Date, Account, Reference Account, Arising, Voucher type).

Select “Auto Numbering”

“Auto Numbering” option allows numbering vouchers by a defined form. For example: “Journal voucher number” is automatically created following form “No.”/mm_yy; in which, mm_yy is month_year of “Date from”. If “Auto Number” is unmarked, you have to manually number vouchers in the period. The system will check the duplicate numbers in a year and prevent saving.

Select “Filter”

There are 3 options for “Filter” field:

    - All Trans.: group all original vouchers that meet filter condition in step 1;
    - Trans with no Batch No.: group all ungrouped original vouchers that meet filter conditions in one voucher;
    - Trans with Batch No.: group all grouped original vouchers that meet filter conditions in one voucher..

Register

Click “Save” to come back to come back to journal voucher list screen (arising amount is 0), select journal voucher and click “Register”

Tasks

Clone the Journal Voucher Register

“Copy” function allows selecting journal vouchers that will be registered in the next period without declaration. Process:

    - Open the function “Journal Voucher Register” or press Choose Period button (when you are in journal voucher screen),
    - Choose “Copy Journal Voucher Register to Next Period”.
    - Type the date of the original Voucher to be copied into “Date from/to” fields.
    - Press “OK”, you will have new Journal Voucher for the next month with the same number rules and filter conditions.

Mass registering

To register series of vouchers: mark vouchers to select then press the icon “Journal Voucher Register” to register.

Mass delete the registering

To delete series of vouchers: mark vouchers to select, then press the icon “Cancel Register”.
