Contract Opening Balance

To businesses having customer balance tracking and balance tracking by contract, this function is used to input contract opening balance, which supports in reporting accounts payable by contract, before using the program.

Entry screen

Field descriptions

Account

The account in which the contact balance is recorded

Contract

The contract of which balance is tracked; extracted from Contract List with contracts having “Balance Tracking” option is activated

Debit Opening Balance

The opening balance of the contract in debit

Credit Opening Balance

The opening balance of the contract in credit

FC Dr. Opening Balance

The opening balance of the contract in debit in foreign currency

FC Cr. Opening Balance

The opening balance of the contract in credit in foreign currency

Description

More description about the balance tracking of this contract

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.

The data importing process:

- Enter "Contract Opening Balance" menu;
- Click the icon to download template file to put data in;
- After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to opening balance file in the system.

Notes of inputting data in template file:
- Unit: Unit is one of the assigned units. The input unit must be one existing in the list with the same length in add (edit) screen;
- Contract: input the contracts existing in the list with the same length in add (edit) screen;
- Debit Balance, Credit Balance: input in number format, 14 digits maximum and 2 decimal digits (standard format);
- FC Debit Balance, FC Credit Balance: input in number format, 12 digits maximum and 2 decimal digits (standard format).

Other notes:
- The system will give error message in case the input string data is longer than the available length of field as one manually input;
- The system will not get data of contracts without balance;
- Check the existence of "Unit, Account": if some code already exists, the system will replace by one in the template file (if the copy type is "Overwrite"). The new codes are certainly added;
- Cannot import data after system or unit data closing (the data closing date after the beginning input date);
- The current user must have the 'Edit" right in "Account Opening Balance" function;
- The current user must have the 'Edit" right in "Access Right by Unit";
- Enable to import data for many units;
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
