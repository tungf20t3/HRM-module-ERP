Fixed Asset Value Adjustment

If you want to adjust depreciation because of huge repair or more parts installed and added value of the asset is also depreciated, please use the function “FA value adjustment”.
This function is the same as creating a new asset but the difference is that depreciation will be grouped to the adjusted asset.
Path: Modules → Fixed Assets Management → Data Entry → Fixed Asset Value Adjustment

Entry screen


Field Descriptions

FA Code

Code of adjusted asset, extracted from “Asset List”

Period / year

Month of adjustment

Capital Code

Sourced capital of original cost increase; capital code is extracted from Capital List.

Reason

Increase reason of original cost, reason code is extracted from increase/decrease reason list.

VC No. / Date

Number of voucher related to this adjustment

FC Code

Currency code is extracted from fixed asset list.

Ex. Rate

Exchange rate is extracted from fixed asset list; editable.

Original Cost / Accumulated Depr. / Current value

The information relating to asset increase value adjustment. Add foreign currency field for assets having foreign currency value.

Remaining Periods

Remaining depreciation periods of asset after adjustment; system-generated but editable. If asset adjustment makes the depreciation time changed, you have to enter it. This field is an important base to redefine monthly depreciation value after adjustment.

Formula: Remaining periods=Total depreciation periods of the asset - (period/year of adjustment - original period/year of asset)

This information is valid to asset depreciated by straight-line method.

Monthly Depr. Adjustment

Monthly depreciation adjustment value will be system-generated following current depreciation regulations.

-If increase: Monthly depreciation adjustment value will be equal to:
Total current value by capital at the beginning of adjustment period
(-) Subtract:
Monthly depreciation by capital in asset original information
(+) Add:
Total monthly depreciation adjustment value by capital in increase adjustment (including one of adjustment period if any)
(-) Subtract:
Total monthly depreciation adjustment value by capital in decrease adjustment (including one of adjustment period if any)

- If decrease: the formula is the same as above but the result in the opposite sign.
Monthly depreciation value is automatically calculated following current depreciation regulations.

This information is valid to asset with depreciation by straight-line method.

Depreciation Value

In case this field is system-calculated and read-only, you cannot edit. It shows periodic depreciation value of the asset after adjustment and the depreciation will be calculated based on this value.
If you want to change this value, you can adjust the value at “Monthly Depr. Adjustment” field only.
This information is valid to asset depreciated by straight-line method.
