Cash Disbursement Allocation to Invoices

“Cash Disbursement Allocation to Invoices” function is used to allocate amount paid to supplier to purchase invoices. Only disbursements, bank debit advices Transaction Type 2, 5 appear to be allocated.

“Cash Disbursement Allocation to Invoices” Screen

Field descriptions

Inv. No.

Number of invoices are not paid or partially paid to supplier, extracted from list of invoices without full paid.

Inv. Date

The date the invoice is created.

FC Code

Code of transaction currency in invoice

Dr. Acct.

Debit account in decrease payable transaction. It is automatically updated based on payable account used to reflect transaction when having invoice. Dr Account in this case cannot be entered but extracted from invoice.

Invoice Amt.

Amount in transaction currency in invoice; automatically updated.

Paid

Amount paid to invoice. It is automatically updated (converted to transaction currency in invoice). .

Payable

Amount will be paid to invoice (in transaction currency). It is automatically updated (in invoice currency).

Payment

Payment in transaction currency of debit memo

Converted Payment

If transaction currency in credit memo is different from that in invoice, Converted Payment field will be updated to have amount converted to transaction currency in invoice. Otherwise; this field will not be updated.

Description

Transaction reasons/ memo, posted to GL as Description field.

Base Curr. Amt.

Amount payable in GL currency. If transaction currency is the same as GL currency, this field will not be displayed. This field’s value equals FC amount multiplied with exchange rate, but editable.

Operations

View and select Disbursement Voucher

When you choose “Allocate disbursement amount to invoices” function, Latest updated Disbursements and Debit Memos in “View Voucher” screen will be displayed.

Allocate Disbursement Amount to Invoices

To allocate amount in Disbursement or Debit Memo to invoices, click <> button .

Filter voucher

In case it is required to filter entered Disbursement, Debit Memo to choose appropriate ones to allocate, click <> button.

View Disbursement and Debit Memo

To view filtered Disbursement, Debit Memo, click <> button or right-click mouse and choose View. Program will display voucher quick view screen.
Format view includes general information of voucher and created user, created date, modified user, and modified date.
Grid view includes information of GL in details.
