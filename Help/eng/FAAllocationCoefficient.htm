Fixed Asset Allocation Coefficient Definition

The function is used when asset depreciation is allocated by many expense accounts, many using departments, many jobs, many expense codes... and allocation coefficient must be defined by account, department, job, expense code.

Entry screen

Field Descriptions

FA Code

Extracted from Fixed Asset List

Depreciation Account

Depreciation account, extracted from Chart of accounts

Expenses Account

Extracted from Chart of accounts

Depreciation Dept.

Department code is extracted from Using Department List in Fixed Assets Management.

Cust. ID

Customer ID, extracted from Customer List

Job ID

Job code, extracted from Job/Contract List in Costing module

Product

Product code, extracted from Item Master, item type 51- Finished goods

Dept.

Department code, extracted from Department List in System module

Contract

Contract code, extracted from Contract List in Accounts Payable

Loan Contract

Loan contract code, extracted from Loan Contract List in Cash Management

Fee

Fee code, extracted from Fee List in Expenses Management

MO No.

Manufacturing order number, extracted from manufacturing order in Advanced Costing module

Free Code 1/2/3

Free code 1/2/3, extracted from free list 1/2/3 in User-defined Field Report

Coefficient

Defined by user in accordance with each account, job, and expense code.
