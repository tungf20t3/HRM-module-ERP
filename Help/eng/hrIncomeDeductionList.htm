Income, Deduction List

One assessable income type may include some income codes.

Entry screen

Field descriptions

Income Type

This is a required field; extracted from “PIT Assessable Income List”.

Nature

Option to define that this code is deduction or income when calculating the assessable income

Tax Threshold

Input amount; effective only when “Nature” is “1 – Income” and PIT type of this income is “Entire” This tax threshold will be value of “Deduction” field in “Income by the flat rate tariff” entry so that assessable income will be equal to taxable income subtracting deduction

Tax Code

Effective only when PIT type of this income is “Entire” (defined in PIT assessable income list); extracted from the scale of flat rate tariff and a required field
