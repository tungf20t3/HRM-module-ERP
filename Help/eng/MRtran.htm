Material Requisition

This voucher is usefull for businesses which had the "Material Requisition" process. Production manager will approve this document before issue item. Material Requisition can be inherited data from manufacturing order. The program will base on the quantity of finish good, semi-product and bill of material to explore to the lowest level such as item.

Material Requisition screen

Fields Description

The major informations (item, qty, product,..) is extracted from manufacturing order, user only input the extra fields (site, job,..) and choose the status to complete the Requisition.

Some common tasks in creating Material Requisition

Extract from MO

In some cases, items in Material Requisition (MR) can be extracted from created Manufacturing Order (MO). Item quantity is calculated by quantity required in MR multiplied by factors defined in Bill of Material (BOM). To extract data from MO, create an MR, click on function “ - Extract Data from MO", and type start date to lookup MOs in MO selecting screen. Choose selected MO and MO lines, key in required quantity and back to previous screen. MR will be shown with equivalent components of the selected items.

Note
- Required Qty will cover the scrap factor defined in BOM.
- Required items are components/ raw materials. If there is semi-product items in MO then, itemts will be explored to the lowest level.
