Employee Card Info Input

Employee card information input is main information in Timecard Management. The information defined in this function is priority on data gathering for workday computation.

Entry screen

Field descriptions

Default Workday

It is required to select a workday with category 1 - Regular (Workday Sign List). If this is not a regular workday, the annual leave calculation will be inaccurate (weekend, holidays will also be subtracted on annual leave).

Default workday is used to replace the workday sign in case of working on non-working days or holidays.

Hours

Number of hours defined in this function is base for workday computation. If this field is blank, the system will refer to "No. of Working Hours per Day" parameter in system for workday computation. Then, workday will be calculated by actual working hours dividing this defined hours.
