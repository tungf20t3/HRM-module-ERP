Advanced Calculation

This is an auto-costing function. All costing steps will be system-calculated from the beginning to the end. Costing in separated steps is used for checking the accuracy of the costing setup. If it is correct, use advanced costing.

Possible steps in advance costing:

    - Calculate quantity of product received in period;
    - Calculate quantity of product manufactured in period;
    - Aggregate arising cost;
    - Post direct cost;
    - Calculate allocation ratio by BOM;
    - Allocate arising cost;
    - Calculate average cost or;
    - Calculate daily average cost or;
    - Calculate FIFO cost;
    - Calculate WIP value;
    - Calculate cost of product.

This function is used for multi-section costing. Each section is opening data for the next one, which consists of steps (tasks) being partially or absolutely the same as ones in some previous sections.

For example:
Section 1 is used to calculate item group 1, allocate cost and update cost
Section 1 is used to calculate item group 2, allocate cost and update cost
Section 1 is used to calculate item group 3, allocate cost and update cost…
