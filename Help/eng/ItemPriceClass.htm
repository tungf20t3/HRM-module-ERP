Item Price Class List

Item price class definition supports in setting up list of discounts by item group in sales. If a discount code defined to be calculated by item group, it is required to select item price class in discount detail definition.

This list is used when your business has discount policy applied to item group. To calculate the discount for each item in the defined group, it is required to select the “Item Price Class” during item definition in “Item List”.
In sales order entry – item details, if the customer is applied discount by item, the system will find and apply the corresponding discount details.

Some tasks in list maintenance

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:

    - Enter Item Price Class List;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to the List.

Notes of inputting data in template file:
- Code: in form of string with 8 characters maximum. One code is unique in this column. If some code already exists in List, the system will replace by one in the template file (if the copy type is "Overwrite"). The new codes are certainly added to List;
- Code cannot be nested each other or consist of system master or operator characters: '`~!@#$%^&*(),;"<>?/,_;
- Description, Other Name: user-defined information, 128 characters maximum.

Other notes:
- If you do not have the "New" right in "Item Price Class List", you cannot import new data from spreadsheet to this list;
- If you do not have the "Edit" right in "Item Price Class List", you cannot edit data in this list from spreadsheet;
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
