Tool & Supply Issue

The information input in this menu is base for tool and supply issue calculating and reporting.

Expiry Date

This is an optional field. The expiry date is the same for rows of the same TS code. The system will give warning message if the "Expiry Date" is before the "Issue Date".

Import data from excel file

The system is enabled to import data from excel spreadsheet to save hours of manual effort.
The data importing process:

    - Enter Tool & Supply Issue;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to grid view of the voucher.

Notes of inputting data in template file:
- Issue Date, Expiry Date: input in date format; Expiry Date must be after or the same as Issue Date;
- Number: manually input, required field;
- Memo: manually input, optional field;
- Employee ID, TS Code: must be one existing in the corresponding list (Employee, Tool and Supply List) with the same length in add (edit) screen in list; employee ID must be one belonging to departments in the assigned units.

The system will give error message in some following cases:
- Data in column containing employee, tool and supply do not exist in the corresponding lists. In this case, the system will indicate the specific invalid cell;
- String data is longer than the available length of field (indefinite error);
- Wrong date format;
- "Expiry Date" is before "Issue Date".

Other notes:
- You must have the "New" right to import new data to voucher detail grid;
- You must have the "Edit" right to edit data in voucher detail grid.
