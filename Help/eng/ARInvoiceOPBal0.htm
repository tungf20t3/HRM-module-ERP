Opening AR Vouchers for Invoices

To businesses having AR controlled by invoice, this function is used to input payment vouchers created before using the AI ERP. These vouchers can be used for invoice allocation then.

Entry screen

Fields descriptions

- If “Currency” is the same as base currency defined in optional parameters, there are following fields only:

    + Amount: received amount in Cash Receipt/ Bank Credit Advice;
    + Allocated Amount: amount has been allocated before; it can be blank;
    + Current Amount = Amount – Allocated Amount; system-calculated and un-editable.

- If “Currency” is different from base currency defined in optional parameters, there are following fields only:

    + FC Amount: received amount in Cash Receipt/ Bank Credit Advice;
    + FC Allocated Amount: amount has been partially allocated before; it can be blank;
    + FC Current Amount = Amount – Allocated Amount; it’s system-calculated and un-editable.
    + Exchange Rate: in Cash Receipt/ Bank Credit Advice;
    + Amount = FC amount * Exchange Rate; system-calculated and un-editable;
    + Allocated Amount: FC allocated amount * Exchange Rate; system-calculated and un-editable;
    + Current Amount = Amount – Allocated Amount; system-calculated and un-editable.
