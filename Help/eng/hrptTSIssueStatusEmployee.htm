Tool & Supply Issue Status by Employee

Filter Condition

Field Descriptions

Department, Position

The system will get department and position information in following priority:
- 1st: from "Working History/Current Working History", Current Official Position with latest "Start Date" to "Report Date";
- then: from "General Information".

Tool & Supply

Tools and supplies input in "Tool and Supply Request by Job Position" by employee in each department and position.

Required Qty

The quantity input in "Tool and Supply Request by Job Position"

Used Qty

Used quantity is equal to issued quantity minus decreased quantity.

Issued Qty

Issued quantity is one input in "Tool and Supply Issue" with issue date before or the same as report date which is before or the same as expiry date.

Decreased Qty

The quantity input in "Tool and Supply Decrease" with decrease date before or the same as report date which is before or the same as expiry date.

Excessive / Insufficient Difference

Excessive / insufficient difference is equal to required quantity minus used quantity;
If the difference is greater than 0, it will be shown in "Insufficient" column. On the contrary, if it is less than 0, it will be shown in "Excessive" column (in positive number).
Report can be filtered by:

    1 – Insufficient;
    2 – Excessive;
    3 – Both (insufficient and excessive diffenrence) ;
    4 – All.
