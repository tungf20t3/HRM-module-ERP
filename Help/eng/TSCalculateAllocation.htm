Tool & Supply Allocation Transaction

Allocation is calculated by the system based on the data and formula defined in tool & supply information entry. However, you can edit the amount with “Adjust monthly allocation" function. Allocation adjustment occurs because current amount is so small and you want to add it to current month allocation.
The result will be saved in a data file. If there is any change, users must recalculate allocation.
Path: Modules → Tools & Supplies Management → Tool & Supply Allocation Transaction

Screen

Descriptions

Straight-line method

Allocation must be periodic (monthly) calculated; allocation value equals original cost or current value divided by periods.

Two times method

The 1st allocation is calculated in allocation date and the 2nd one is ending allocation date or broken date of tool & supply.

Note: program supports user with allocation by date and by original cost or current value method; this declaration is created in “Optional parameter declaration”.
