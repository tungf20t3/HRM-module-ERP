Timekeeping Data Import

Common tasks

Download Template File

Click to download template file which has not been exported from timekeeping machine and must be edited. There are following fields:

    - Month/Year: month/year of timekeeping data imported to server;
    - Employee Card Number: one input in “Employee Card Info Input";
    - Date &Time: must be in dd/mm/yyyy hh:mm:ss format. For example: 03/02/2013 16:57:00;
    - Type: 1 is Time in, 2 is Time out. In case this field is blank, time in/out will be system-generated.

Import Data from File

Click to import timekeeping data to server;
Click “Select…” to choose timekeeping data file;
On “Timekeeping Data Import” window, select copy type 1 – Overwrite if you want to update all timekeeping data; otherwise, if you select type 0 – No, only new rows will be inserted to the available data.

Note
01. The imported data have been checked following filter condition. In case all data in excel file satisfy the filter condition but there also be errors, check the timekeeping date.
02. If the importing date is not within the effective period (from/to) of card number defined in “Employee Card Info Input”, the data will not be imported.
