Appraisal Reports

Performance Appraisal Form

The data for this report is retrieved from “Performance Appraisal Information”.

Filter condition

Performance Appraisal Template

The same as “Performance Appraisal Form” but it is the blank form for the appraiser and appraise to fill in the appraisal.

Filter condition

Some special fields

Date of Appraisal

After the date of appraisal is defined in the filter condition, the system will update it to the performance appraisal template and calculate the years of service and years in the current position of each employee.
Refer to “Performance Appraisal Information” for more details

Employee Appraisal Summary

This report will be shown with the data from “Performance Appraisal Information”.

Filter condition

Some special fields

Self-Score/ Appraiser Score

These scores are ones from “Total Self-Score” and “Total Appraiser Score” in “Performance Appraisal Information” in the period/year in accordance with ones in filter condition.

Multi-period Appraisal Summary

This report will be shown with the data from “Performance Appraisal Information”.

Filter condition

Some special fields

Average Score

This is average score of all periods with score greater than 0.

Period

The score in Period 1, 2 … will be shown as “Total Self-Score” or “Total Appraiser Score” depending on the “Appraisal Type” in filter condition.

Appraisal Progress

This report will be shown with the data from “Performance Appraisal Information”. Only employees with performance appraisal information in accordance with filter condition will be shown in report.

Filter condition

Some special fields

“Self-Score %”

This is percentage of the scored criteria (level-2 criteria and level-1 criteria without sub-criteria) on total criteria (level-2 criteria and level-1 criteria without sub-criteria).

Note: consider the self-score of each employee

“Appraiser Score %”

This is percentage of the scored criteria (level-2 criteria and level-1 criteria without sub-criteria) on total criteria (level-2 criteria and level-1 criteria without sub-criteria).

Note: consider the appraiser score of each employee

“Status”

The status of the performance appraisal form
