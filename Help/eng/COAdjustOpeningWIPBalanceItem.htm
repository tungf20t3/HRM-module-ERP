Adjust Opening Balance of WIP by Item

Entry Screen

Field Descriptions

Cost Factor

Code of cost factor WIP detailed to product

Product

Code of product WIP opening adjusted

Department

Code of production line, workshop in case product WIP by line, workshop. This is a required field if cost is calculated by department in system parameter.

MO Number

Number of MO WIP in case product WIP in MO. This is a required field if cost is calculated by MO in system parameter.

FC Amount/ Amount

WIP amount in foreign currency and amount in GL detailed by cost factor
