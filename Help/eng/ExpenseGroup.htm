Expense Group List

Expense group list is used to classify expenses in main groups. Expenses having some properties or characteristics in common will be put in a group.
These groups can be classified in group type. A group type consists of groups having the same classified properties. There are now 3 group types: 1, 2 & 3.

For example: Group type 1 refers to “Purchase expense”.


Expense grouping supports in data managing and reporting by expense group. If you consider 3 group types as 3 sides of analysis, in summary reports, it can be reversed to have different analysis reports.

Some tasks in list maintenance

Get data from excel file

The system is enabled to get data from excel spreadsheet to save hours of manual effort.
The data getting process:

    - Enter Expense Group List;
    - Click the icon to download template file to put data in;
    - After keying data in the spreadsheet, to up data to the system, click the icon , select the file, the system will get data from this file to the List.

Notes of inputting data in template file:
- Group Type: accepted 1 in 3 options (1, 2 or 3);
- Group Code: in form of string with 8 characters maximum. One code is unique in this column. If some code already exists in List, the system will replace by one in the template file (if the copy type is "Overwrite"). The new codes are certainly added to List;
- Group code cannot be nested each other or consist of system master or operator characters: '`~!@#$%^&*(),;"<>?/,_;
- Group Description, Other Name: user-defined information, 128 characters maximum.

Other notes:
- If you do not have the "New" right in "Expense Group List", you cannot import new data from spreadsheet to this list;
- If you do not have the "Edit" right in "Expense Group List", you cannot edit data in this list from spreadsheet;
- Cannot import data from spreadsheet with macro;
- Only support for spreadsheet in Excel 2007 or higher.
