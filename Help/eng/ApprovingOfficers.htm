Approving Officers

Approving Officers are declared to limit competence to approve PR or PO.

“Approving Officer” screen

Field descriptions

Type

Type of voucher being approved (declared by program).

Dept.

Department ID; extracted from “Department List”.

FC

Foreign currency code; extracted from “Currency List”.

Amount

Amount that user has right to approve. If there is no declaration (amount is 0), it means unlimited amount to be approved.

Status

Status, select 1 – Active or 0 – Inactive.
