Cost Factor Type List

Path: Modules → Advanced Costing → List Maintenance → Cost Factor Type List


Cost factor type is used to define different finish rates of period-end WIP detailed by cost factor.


For example:
Finish rates of raw material factors are usually 100%, one of labor cost is 50% and there is no closing WIP for production overhead.

When the closing WIP quantity of product is input by factor type, with quantity and finish rate, the system will generate converted quantity of finished products. Based on this quantity, the system will generate closing WIP value of product.
“Product quantity by factor type report” can be filtered by factor type.
