Employee Information Query

This function is used to extract necessary employee information which cannot be viewed in standard reports. The report forms are user-defined.

Employee Information

Employee information is used in filter condition and user-defined report forms. Just drag and drop to add fields to filter condition or user-defined report forms.

Filter List

Filter Condition Information

There are many ways to set up filter condition by an information.

For example:
If you want to have list of female employees with date of birth from 01 Jan 1990, or single, male employees, you can enter the filter condition rows (filter condition, field name, filter comparison and filter value) as following:
- And Open Bracket + Gender + Equal + Female;
- And Closed Bracket + Date of Birth + Greater or Equal + 01/01/1990;
- Or Open Bracket + Gender + Equal + Male;
- Or Closed Bracket + Marital Status + Equal + Single
Or filter condition rows:
- None + Gender + Not Equal + Male;
- And + Date of Birth + Greater or Equal + 01/01/1990;
- Or Open Bracket + Gender + Equal + Male;
- Or Closed Bracket + Marital Status + Equal + Single

Special tasks

New

If there are available filter conditions of default report form, click to remove them and add new filter conditions.

Copy

If you want to use available template, click to copy it. The template can be used by other users when the “Public Template” option is checked. Otherwise, it can only be used by the creator.

Save/ Save as

You can save or save as template if there is any change in filter condition.

Delete

Click to delete current template.

Set as Default

An available template can be set as default one for each user or not when you click icon.

Template

Employee List

The query template can be user-defined following your requirements. Click the icon to set up your template;
Drag and drop employee information to create query templates.

Click to view query information.

For example:
If you want to view list of employees with company award by dated sign, drag and drop Company and Date Signed to employee list fields. You cannot view date from/to, gender,... if you do not add them to filter condition. The query information can be exported to excel spreadsheet or sent you via your email.

Note:
You must be assigned to view employee information.
