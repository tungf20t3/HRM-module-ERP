Calculate Period-end WIP Cost by Physical Counting

By 2 menus “Quantity of WIP Closing” and “Input Closing Inventory”, program has data of item WIP closing. Program will use price of closing inventory to calculate Period-End WIP cost by physical counting.
To test the result of this calculation, enter reports:
- Period-end Raw Material Cost by Physical Counting
This calculation can be ignored in case it is not detailed by materials.
