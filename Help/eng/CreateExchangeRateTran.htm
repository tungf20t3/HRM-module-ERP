Exchange Rate Difference Entry

Path: General → Closing Transaction → Exchange Rate Differennce Entry
Create the exchange rate difference entry at the end of period

Field descriptions

Select

Check-box option to select the exchange rate entries to be evaluated at the end of period

No./ Description/ Account/ Currency/ Ex. Rate Account

See “Exchange Rate Difference Entry Definition” guides for more details

Some basic tasks

Select/ Unselect

Click on “Select” check-box to select or unselect the entry

Select all

Click the arrow above () to select all the entries in list

Unselect all

Click the arrow () again to unselect all selected entries in list

Create Exchange Rate Entry

Click “

Create exchange rate entry ” to system-generate entries in ledger

Delete Exchange Rate Entry

To delete the generated entries in ledger, select the desired entries, click “

Delete exchange rate entry”.

Note:
+ If the parameter "Type of Exchange Rate Revaluation for AR/AP Account" is “Single Side Revaluation” and AR-AP account is in “List of AR/AP Account with FC Revaluation in Debit Balance”, only exchange rate of customers with closing debit balance will be revaluated;
+ If the parameter "Type of Exchange Rate Revaluation for AR/AP Account" is “Single Side Revaluation” and AR-AP account is in “List of AR/AP Account with FC Revaluation in Credit Balance”, only exchange rate of customers with closing credit balance will be revaluated.
