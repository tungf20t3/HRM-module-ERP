Transfer Insurance Information to Next Year

This function is used to transfer "No. of Employees" and "Wage Fund" used in "Employees Joining Insurance" to next year.

Note: Only transfer data of the units that the current user has the access right.

Number of employees

Number of employees transferred = Opening number of employees + Labor increase - Labor decrease;

- Opening number of employees is firstly input in "Insurance Information Summary Input" or transferred from previous year.
- Labor increase:

    + “No. of Employees Increase SI, HI”: count records (in "Insurance Information Changes Input") with "SI Change Type" 1 - Labor Increase and SI Rate Increase, without "Date to";
    + “No. of Employees Increase UI”: count records (in "Insurance Information Changes Input") with "SI Change Type" 1 - Labor Increase or 7 - UI Increase and UI Rate Increase, without "Date to".

- Labor decrease:

    + “No. of Employees Decrease SI, HI”: count records (in "Insurance Information Changes Input") with "SI Change Type" 2 - Labor Decrease and SI Rate Decrease, without "Date to";
    + “No. of Employees Increase UI”: count records (in "Insurance Information Changes Input") with "SI Change Type" 2 - Labor Decrease or 8 - UI Decrease and UI Rate Decrease, without "Date to".

Total wage fund

Total wage fund transferred = Opening wage fund + Salary increase - salary decrease;

- Opening wage fund is input in "Insurance Information Summary Input";
- Salary increase: sum of all records with "SI Change Type" increase, without "Date to" in the transfer year;
- Salary decrease: sum of all records with "SI Change Type" decrease, without "Date to" in the transfer year, report times before the current one.
