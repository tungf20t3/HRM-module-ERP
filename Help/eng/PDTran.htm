Field descriptions – Grid view

Item code

Code of received item; extracted from “Item Master” if Purchase Receipt is created new or from declaration in PO if data in Purchase Receipt is extracted from PO.

Item Name

Name of item received

UOM

Unit of measurement of item received, extracted from “Item Master”

Site ID

Site receiving items, extracted from “Site List”

Location ID

Location where items are stored, extracted from “Location List”

Lot ID

Number of lot of received items

Quantity

Quantity of received items required field.

QC

This option shows that items need checking before receipt or not. If there is tick on this option, program will list items selected when creating QA Transaction and status of these items will be “Open” after saving voucher. Otherwise, status will be “Receipt”.

PO

Number of PO; shown only when data of Purchase Receipt is extracted from PO.

PO Line

Number of line in PO; shown only when data of Purchase Receipt is extracted from PO.

(FC) Price

Received unit price of item in declared foreign currency; appear only when currency field in format view is foreign currency.

(FC) Amount

Received amount of item in defined foreign currency; appear only when currency field in format view is foreign currency.

Price

Unit price of received items in VND; used to replace (FC) Price field in case format view - VND.

Amount

Amount of received items in VND; used to replace Price (USD) field in currency field in format view is VND.

Status

Status of received items is automatically processed by program. There are 02 statuses: “Open” and “Receipt”.

Invoice Qty

Quantity of received items is entered in “Purchase Invoice” screen.

Job

Job, project, work…relating to receipt materials

Some common operations in Purchase Receipt entry

Create Purchase Receipt by Extract data from PO

Create new Purchase Receipt, click " - Extract Data”, select PO type (0 – Domestic Purchase Order, 1 – Imported Purchase Order) Input the date from which the PO is selected, then click to filter PO. Only PO in “Approved” or “Receipt” status of the selected supplier appears in the screen. Select the required PO, input the receipt quantity and exit the PO list screen. The selected rows in the PO list will be systematically updated to Purchase Receipt.
